Section 1.3.3

Discovering Invariance

An interesting thing about such transformations, that we have demonstrated moving from infinitesimal to finite transforms, is that they can go both ways.  This will prove very useful…

Above we found the finite level transformation equations for SO(2):

\begin{equation}\left( {\begin{array}{*{20}{c}}   {x'} \\   {y'} \end{array}} \right) = \left( {\begin{array}{*{20}{c}}   {\cos \theta }&{ - \sin \theta } \\   {\sin \theta }&{\cos \theta } \end{array}} \right)\cdot \left( {\begin{array}{*{20}{c}}   x \\   y \end{array}} \right)\notag\end{equation}

These emerged from repeated application of inching along with this infinitesimal (“Inchie”) level transformation:

\begin{equation}\left( {\begin{array}{*{20}{c}}   {x'} \\   {y'} \end{array}} \right) = \left( {\begin{array}{*{20}{c}}   1&{ - \varepsilon } \\   \varepsilon &1 \end{array}} \right)\cdot \left( {\begin{array}{*{20}{c}}   x \\   y \end{array}} \right)\notag\end{equation}

We now discuss the formal way of accomplishing the reverse, developed by Lie.  We will use modern notation from Olver's 1986/1995 texts that is common throughout 20th century treatments. At the finite level we can think of a continuous transformation group formally like this:

\begin{equation}\left( {x',y'} \right) = {g_\varepsilon }\emptyset \left( {x,y} \right) = \Psi \left( {\varepsilon ,\left( {x,y} \right)} \right)\notag\end{equation}

with the “dot” symbolically stating that $\left( {x,y} \right)$ is acted upon by an element of the group whose symbol is ${g_\varepsilon }$ and the rightmost expression declaring the operation results in a set of functional equations $\Psi $ that take $\left( {x,y} \right)$ and produce $\left( {x',y'} \right)$.  The subscript on the middle expression's symbol ${g_\varepsilon }$ and inside the argument list of $\Psi $ indicates that there is a parameter $\varepsilon $ that is part of the group's transformation equations.  Earlier we used the angle $\theta $ as this parameter, but soon we will take an infinitesimal limit so use $\varepsilon $ commonly associated with infinitesimal parameters.  Possessing only a single parameter $\varepsilon $ these are called one-parameter groups.

To take us to the infinitesimal level we let group parameter $\varepsilon $ become infinitesimal and expand the function $\Psi $ around the zero of the parameterization:

\begin{equation}\left( {x',y'} \right) = \Psi \left( {\varepsilon ,\left( {x,y} \right)} \right) = {\left. {\Psi \left( {\varepsilon ,\left( {x,y} \right)} \right)} \right|_{\varepsilon  = 0}} + {\left[ {\frac{{d\Psi }}{{d\varepsilon }}} \right]_{\varepsilon  = 0}}\left( {\varepsilon  - 0} \right) + O\left( {{\varepsilon ^2}} \right)\notag\end{equation}

But the first term has to be $\left( {x,y} \right)$ so:

\begin{equation}\left( {x',y'} \right) \approx \left( {x,y} \right) + \varepsilon {\left[ {\frac{{d\Psi }}{{d\varepsilon }}} \right]_{\varepsilon  = 0}}\notag\end{equation}

This bears a resemblance to derivative the method we used above to incrementally translate Inchie. 

It proves useful to define the tangent vector to the transformation-induced path-curve at the initial point:

\begin{equation}{\left. V \right|_{\Psi \left( {\varepsilon ,\left( {x,y} \right)} \right)}} \equiv \frac{{d\Psi \left( {\varepsilon ,\left( {x,y} \right)} \right)}}{{d\varepsilon }}\notag\end{equation} Then \begin{equation}{\left. V \right|_{\left( {x,y} \right)}} \equiv {\left. {\frac{{d\Psi \left( {\varepsilon ,\left( {x,y} \right)} \right)}}{{d\varepsilon }}} \right|_{\varepsilon  = 0}}\notag\end{equation}

Expanding $\Psi $ in the small parameter $\varepsilon $ we reproduce Olver (1.12):

\begin{equation}{\left. V \right|_{\left( {x,y} \right)}} = \frac{d}{{d\varepsilon }}\left\{ {{{\left. {\Psi \left( {\varepsilon ,\left( {x,y} \right)} \right)} \right|}_{\varepsilon  = 0}} + \varepsilon {{\left. {\frac{{d\Psi }}{{d\varepsilon }}} \right|}_{\varepsilon  = 0}} + O\left( {{\varepsilon ^2}} \right)} \right\} = {\left. {\frac{{d\Psi }}{{d\varepsilon }}} \right|_{\varepsilon  = 0}}\notag\end{equation}

Please note that this technique of taking a derivative of a series expansion with respect to the series parameter and then taking the limit as that parameter goes to zero is an intuitive and very effective way of finding the term in the series with $\varepsilon $ to the first power.  For this reason this term is often called the local linear approximation.

Then note that this is NOT the local linear approximation (LLA) to the function $\Psi $, but the LLA for the change in the function $\Psi $ moving away from the zeroeth order term $\left( {x,y} \right)$.  This is Lie's infinitesimal approximation system in action.

Applied to our running example this looks like:

\begin{equation}{V_R} \equiv \frac{d}{{d\theta }}{\left[ {\left( {\begin{array}{*{20}{c}}   {\cos \theta }&{ - \sin \theta } \\   {\sin \theta }&{\cos \theta } \end{array}} \right)\cdot \left( {\begin{array}{*{20}{c}}   x \\   y \end{array}} \right)} \right]_{\theta  \to 0}}\notag\end{equation}

Technically we should take the derivative before taking the limit on small parameter as the operation derives from a Taylor series:

\begin{equation}{V_R} = {\left[ {\left( {\begin{array}{*{20}{c}} { - \sin \theta }&{ - \cos \theta } \\ {\cos \theta }&{ - \sin \theta } \end{array}} \right)\cdot \left( {\begin{array}{*{20}{c}} x \\ y \end{array}} \right)} \right]_{\theta \to 0}} = \left[ {\left( {\begin{array}{*{20}{c}} { - \varepsilon }&{ - 1} \\ 1&{ - \varepsilon } \end{array}} \right)\cdot \left( {\begin{array}{*{20}{c}} x \\ y \end{array}} \right)} \right]{|_{\varepsilon \to 0}} = \left( {\begin{array}{*{20}{c}} { - y} \\ x \end{array}} \right) \notag\end{equation}

However, we may also take the perspective that all we really need is the leading expansion term before taking the derivative, yielding

\begin{equation}{V_R} = \frac{d}{{d\varepsilon }}{\left[ {\left( {\begin{array}{*{20}{c}}   {1 - \frac{1}{2}{\varepsilon ^2}}&{ - \varepsilon } \\   \varepsilon &{1 - \frac{1}{2}{\varepsilon ^2}} \end{array}} \right)\cdot \left( {\begin{array}{*{20}{c}}   x \\   y \end{array}} \right)} \right]_{\varepsilon  \to 0}} = \left[ {\left( {\begin{array}{*{20}{c}}   { - \varepsilon }&{ - 1} \\   1&{ - \varepsilon } \end{array}} \right)\cdot \left( {\begin{array}{*{20}{c}}   x \\   y \end{array}} \right)} \right]{|_{\varepsilon  \to 0}} = \left( {\begin{array}{*{20}{c}}   { - y} \\   x \end{array}} \right)\notag\end{equation}

The tangent vector (LLA) expressed in component form looks like:

\begin{equation}{V_R} =  - y\frac{\partial }{{\partial x}} + x\frac{\partial }{{\partial y}}\notag\end{equation}

Can we deduce some kind of invariant from this transformation? Recall we must impose change in order to find things that don't vary.  The most common way of imposing change in mathematics is to take the derivative of a generic function-field and demand it vanish, signally invariance under that operation.  Proceeding, we posit a generic function $\zeta \left( {x,y} \right)$ and demand

\begin{equation}{V_R}\zeta \left( {x,y} \right) =  - y\frac{{\partial \zeta }}{{\partial x}} + x\frac{{\partial \zeta }}{{\partial y}} = 0\notag\end{equation} or \begin{equation}y\frac{{\partial \zeta }}{{\partial x}} = x\frac{{\partial \zeta }}{{\partial y}}\notag\end{equation}

Let's now determine the function that satisfies this.  We take a generic function to be $\zeta \left( {x,y} \right)$ and insist it be invariant when operated on by our vector generator, which requires it to satisfy

\begin{equation}y\frac{{\partial \zeta }}{{\partial x}} = x\frac{{\partial \zeta }}{{\partial y}}\notag\end{equation}

There many ways to solve for $\zeta \left( {x,y} \right)$ - most usefully the Method of Characteristics described in Section ?? – but for such a simple problem we may assume an ansatz such as

\begin{equation}\zeta \left( {x,y} \right) = X\left( x \right)Y\left( y \right)\notag\end{equation} \begin{equation}yYX' = xXY'\notag\end{equation} so \begin{equation}\frac{{X'}}{{xX}} = \frac{{Y'}}{{yY}} = c = constant\notag\end{equation} So \begin{equation}\frac{{X'}}{X} = cx \to \ln X = \frac{1}{2}c{x^2} + b \to X = \exp \left( {\frac{1}{2}c{x^2} + b} \right) = A\exp \left( {\frac{1}{2}c{x^2}} \right)\notag\end{equation} similarly \begin{equation}Y = B\exp \left( {\frac{1}{2}c{x^2}} \right)\notag\end{equation}

then reconstituting our function and redefining some constants

\begin{equation}\zeta \left( {x,y} \right) = C\exp \left( {{x^2} + {y^2}} \right) = invariant\notag\end{equation}

This solution is an invariant function under the operation of the rotation group transformation.  It also says that the combination ${x^2} + {y^2} \equiv {r^2}$ is invariant, which of course generates circles.  It is no surprise that an invariant of the rotation group is a circle.

Reviewing our work, we started by proposing an infinitesimal transformation

\begin{equation}\left( {\begin{array}{*{20}{c}}   {x'} \\   {y'} \end{array}} \right) = \left( {\begin{array}{*{20}{c}}   1&{ - \varepsilon } \\   \varepsilon &1 \end{array}} \right)\cdot \left( {\begin{array}{*{20}{c}}   x \\   y \end{array}} \right)\notag\end{equation}

and showed how repeated applications (inching along) led to the rotation group of symmetries, which then led to the rotation vector generator under which circle functions are invariant.  Returning to the definition of a group, this symmetry group transformation "takes" a "solution" to the PDE, namely one point $\left( {x,y} \right)$, to another "solution" $\left( {x - \varepsilon y,\varepsilon x + y} \right)$. 

So we have shown the intimate relationship between symmetry and invariance, mediated by group transformations.  However, since this example does not represent a dynamical physical system we cannot really identify a conserved quantity. That will come later.
