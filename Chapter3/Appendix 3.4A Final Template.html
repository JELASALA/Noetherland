Appendix 3.4A

Heat Equation Isovectors via Harrison, et. al., Proces

From Section 3. 4.1, we have for the $LHS\left(A\right)= RHS\left( A \right)$ equation:

\begin{equation}LHS\left( A \right) =  - wdX \wedge dt - dT \wedge du + wdT \wedge dx + dU \wedge dt - Wdx \wedge dt\notag\end{equation} \begin{equation}RHS\left( A \right) = \Lambda _1^1du \wedge dt - \Lambda _1^1wdx \wedge dt + \Lambda _2^1dx \wedge du + \Lambda _2^1dt \wedge dw\notag\end{equation} where the coefficients appearing in the RHS must be zero-forms (functions) to match the degree from the LHS.  For simplicity let’s identify $\Lambda _1^1 = a\left( {x,t} \right)$ and $\Lambda _2^1 = b\left( {x,t} \right)$. \begin{equation}LHS\left( A \right) =  - wdX \wedge dt - dT \wedge du + wdT \wedge dx + dU \wedge dt - Wdx \wedge dt\notag\end{equation} \begin{equation}RHS\left( A \right) = {\text{a}}du \wedge dt - {\text{a}}wdx \wedge dt + {\text{b}}dx \wedge du + bdt \wedge dw\notag\end{equation} Repeating with the $LHS\left(B\right)= RHS\left( B \right)$  we find: \begin{equation}LHS\left( B \right) = dT \wedge dw + dX \wedge du - dU \wedge dx - dW \wedge dt\notag\end{equation} \begin{equation}RHS\left( B \right) = \Lambda _1^2du \wedge dt - \Lambda _1^2wdx \wedge dt + \Lambda _2^2dx \wedge du + \Lambda _2^2dt \wedge dw\notag\end{equation} We can identify and define $\Lambda _1^2 = A\left( {x,t} \right)$ and $\Lambda _2^2 = B\left( {x,t} \right)$ as they must be zero-forms: \begin{equation}LHS\left( B \right) = dT \wedge dw + dX \wedge du - dU \wedge dx - dW \wedge dt\notag\end{equation} \begin{equation}RHS\left( B \right) = {\text{A}}du \wedge dt - Awdx \wedge dt + {\text{B}}dx \wedge du + Bdt \wedge dw\notag\end{equation} Beginning with $LHS(A)$we need to insert the full differentials of the unknown isovector coefficients, except for terms that wedge out to zero:. \begin{equation}\begin{split}LHS\left( A \right) = & - w\left( {{\partial _x}Xdx + {\partial _u}Xdu + {\partial _w}Xdw} \right) \wedge dt - \left( {{\partial _x}Tdx + {\partial _t}Tdt + {\partial _w}Tdw} \right) \wedge du \\ & + w\left( {{\partial _t}Tdt + {\partial _u}Tdu + {\partial _w}Tdw} \right) \wedge dx \left( {{\partial _x}Udx + {\partial _u}Udu + {\partial _w}Udw} \right) \wedge dt - Wdx \wedge dt\end{split}\notag\end{equation} Equating this to the $RHS(A)$ we can match coefficients of each (independent) 2-form elements we obtain some determining equations: \begin{equation} \begin{split} dx\wedge dt:& - w{\partial _x}X - w{\partial _t}T + {\partial _x}U - W = - aw\\ du \wedge dt:& - w{\partial _u}X + {\partial _t}T + {\partial _u}U = a\\ dw \wedge dt:& - w{\partial _w}X + {\partial _w}U = - b\\ du \wedge dx:& + {\partial _x}T + w{\partial _u}T = - b\\ dw \wedge dx:& + w{\partial _w}T = 0\\ dw \wedge du:& - {\partial _w}T = 0 \end{split} \notag\end{equation}

One conclusion we can immediately draw is that at most $T = T\left( {x,t,u} \right)$.  Moving to the second equation:

\begin{equation}LHS\left( B \right) = dT \wedge dw + dX \wedge du - dU \wedge dx - dW \wedge dt\notag\end{equation}

Expanding as above:

\begin{equation}\begin{split}LHS\left( B \right) = & \text{ }\text{ }\left( {{\partial _x}Tdx + {\partial _t}Tdt + {\partial _u}Tdu} \right) \wedge dw + \left( {{\partial _x}Xdx + {\partial _t}Xdt + {\partial _w}Xdw} \right) \wedge du\\ & - \left( {{\partial _t}Udt + {\partial _u}Udu + {\partial _w}Udw} \right) \wedge dx - \left( {{\partial _x}Wdx + {\partial _u}Wdu + {\partial _w}Wdw} \right) \wedge dt\end{split}\notag\end{equation} Matching coefficients with the $RHS(B)$ as above we obtain: \begin{equation} \begin{split} dx \wedge dt:& + {\partial _t}U - {\partial _x}W =  - Aw\\ du \wedge dt:& - {\partial _t}X - {\partial _u}W = A\\ dw \wedge dt:& - {\partial _t}T - {\partial _w}W =  - B\\ du \wedge dx:& - {\partial _x}X - {\partial _u}U =  - B\\ dw \wedge dx:& - {\partial _x}T - {\partial _w}U = 0\\ dw \wedge du:& - {\partial _u}T + {\partial _w}X = 0 \end{split} \notag\end{equation}

Working down from the first set and eliminating the functions $a,b,A\;{\text{and}}\;B$ as we go along, we can combine them pairwise to produce the following set of determining equations:

\begin{equation}W - {\partial _x}U + w\left( {{\partial _x}X - {\partial _u}U} \right) + {w^2}{\partial _u}X = 0 \tag{a}\label{deA}\end{equation} \begin{equation}{\partial _x}T - {\partial _w}U + w\left( {{\partial _w}X + {\partial _u}T} \right) = 0 \tag{b}\label{deB}\end{equation} \begin{equation}{\partial _w}T = 0 \tag{c}\label{deC}\end{equation} \begin{equation}{\partial _t}U - {\partial _x}W - w\left( {{\partial _t}X + {\partial _u}W} \right) = 0 \tag{d}\label{deD}\end{equation} \begin{equation}{\partial _x}X - {\partial _t}T + {\partial _u}U - {\partial _w}W = 0 \tag{e}\label{deE}\end{equation} \begin{equation}{\partial _x}T + {\partial _w}U = 0 \tag{f}\label{deF}\end{equation} \begin{equation}{\partial _u}T - {\partial _w}X = 0 \tag{g}\label{deG}\end{equation}

Begin based loosely on general guidance by Harrison.  Take ${\partial _w}$ of \eqref{deF} and \eqref{deG} equations and apply \eqref{deC}:

\begin{equation}{\partial _{xw}}T + {\partial _{ww}}U = 0 \to U = {c_1}\left( {x,t,u} \right) + w{c_2}\left( {x,t,u} \right)\notag\end{equation} \begin{equation}{\partial _{uw}}T - {\partial _{ww}}X = 0 \to X = {b_1}\left( {x,t,u} \right) + w{b_2}\left( {x,t,u} \right)\notag\end{equation} Look at \eqref{deB}: \begin{equation} - {\partial _x}T + {\partial _w}U - w\left( {{\partial _w}X + {\partial _u}T} \right) = 0\notag\end{equation} Take$\;{\partial _w}$: \begin{equation} - {\partial _{xw}}T + {\partial _{ww}}U - \left( {{\partial _w}X + {\partial _u}T} \right) - w\left( {{\partial _{ww}}X + {\partial _{uw}}T} \right) = 0\notag\end{equation} \begin{equation} - {\partial _{xw}}T + {\partial _{ww}}U - \left( {{\partial _w}X + {\partial _u}T} \right) - w\left( {{\partial _{ww}}X + {\partial _{uw}}T} \right) = 0{\text{\;}}\notag\end{equation} We obtain                       \begin{equation}{\partial _w}X + {\partial _u}T = 0\notag\end{equation} But \eqref{deG} tells us                      \begin{equation} - {\partial _u}T + {\partial _w}X = 0\notag\end{equation} so summing:                          \begin{equation}{\partial _w}X = 0 \to {b_2} = 0\notag\end{equation} differencing:              \begin{equation}{\partial _u}T = 0 \to T = T\left( {x,t} \right)\notag\end{equation} Apply these to \eqref{deB}: \begin{equation} - {\partial _x}T + {\partial _w}U - w\left( {{\partial _w}X + {\partial _u}T} \right) = 0\notag\end{equation} producing                      \begin{equation} - {\partial _x}T + {\partial _w}U = 0\notag\end{equation} But also \eqref{deF}:                       \begin{equation}{\partial _x}T + {\partial _w}U = 0\notag\end{equation} so we also find

\begin{equation}{\partial _w}U = 0 \to {c_2} = 0\notag\end{equation} and \begin{equation}{\partial _x}T = 0 \to T = T\left( t \right)\notag\end{equation} Collecting our learnings: \begin{equation}T = T\left( t \right)\notag\end{equation} \begin{equation}X = {b_1}\left( {x,t,u} \right)\notag\end{equation} \begin{equation}U = {c_1}\left( {x,t,u} \right)\notag\end{equation} Applying learnings into our DEs leaves only three: \begin{equation} - W + {\partial _x}U - w\left( {{\partial _x}X - {\partial _u}U} \right) - {w^2}{\partial _u}X = 0\tag{\ref{deA}}\end{equation} \begin{equation} - {\partial _t}U + {\partial _x}W + w\left( {{\partial _t}X + {\partial _u}W} \right) = 0\tag{\ref{deD}}\end{equation} \begin{equation}{\partial _x}X - {\partial _t}T + {\partial _u}U - {\partial _w}W = 0\tag{\ref{deE}}\end{equation}

Inserting our functional knowledge of the vector coefficients yields

\begin{equation} - W + {\partial _x}{c_1} - w\left( {{\partial _x}{b_1} - {\partial _u}{c_1}} \right) - {w^2}{\partial _u}{b_1} = 0\tag{\ref{deA}}\end{equation} \begin{equation} - {\partial _t}{c_1} + {\partial _x}W + w\left( {{\partial _t}{b_1} + {\partial _u}W} \right) = 0\tag{\ref{deD}}\end{equation} \begin{equation}{\partial _x}{b_1} - {\partial _t}T + {\partial _u}{c_1} - {\partial _w}W = 0\tag{\ref{deE}}\end{equation} Now \eqref{deE} tells us $W$ must be at most linear in $w$, since all other terms are independent of $w$.  Let \begin{equation}W = {d_1}\left( {x,t,u} \right) + w{d_2}\left( {x,t,u} \right)\notag\end{equation} Insert in \eqref{deA} \begin{equation} - {d_1} - w{d_2} + {\partial _x}{c_1} - w\left( {{\partial _x}{b_1} - {\partial _u}{c_1}} \right) - {w^2}{\partial _u}{b_1} = 0\tag{\ref{deA}}\end{equation} We can identify “monomials” in powers of w: \begin{equation} - {d_1} + {\partial _x}{c_1} = 0\notag\end{equation} \begin{equation} - {d_2} - \left( {{\partial _x}{b_1} - {\partial _u}{c_1}} \right) = 0\notag\end{equation} \begin{equation}{\partial _u}{b_1} = 0 \to {b_1} = {b_1}\left( {x,t} \right)\notag\end{equation} Look at \eqref{deD} \begin{equation} - {\partial _t}{c_1} + {\partial _x}\left( {{d_1} + w{d_2}} \right) + w\left( {{\partial _t}{b_1} + {\partial _u}\left( {{d_1} + w{d_2}} \right)} \right) = 0\notag\end{equation} We can identify more monomials in powers of w: \begin{equation} - {\partial _t}{c_1} + {\partial _x}{d_1} = 0\notag\end{equation} \begin{equation}{\partial _x}{d_2} + {\partial _t}{b_1} + {\partial _u}{d_1} = 0\notag\end{equation} \begin{equation}{\partial _u}{d_2} = 0 \to {d_2} = {d_2}\left( {x,t} \right)\notag\end{equation}

Relook at \eqref{deE} with the functional form of W inserted:

\begin{equation}{\partial _x}{b_1} - {\partial _t}{A_1} + {\partial _u}{c_1} - {d_2} = 0\notag\end{equation}

We now collect “second generation” DEs, of which there are five now, having exploited linear independence of coefficients of various powers of “independent” variables (w used so far), a common practice for determining equations but far less ambiguous than prolongations in the Vector Approach which give us derivatives of the dependent variable to rely on.  Here derivatives are considered independent (w in this case).

\begin{equation} - {d_1} + {\partial _x}{c_1} = 0 \tag{i}\label{de1}\end{equation} \begin{equation} - {d_2} - \left( {{\partial _x}{b_1} - {\partial _u}{c_1}} \right) = 0\tag{ii}\label{de2}\end{equation} \begin{equation} - {\partial _t}{c_1} + {\partial _x}{d_1} = 0\tag{iii}\label{de3}\end{equation} \begin{equation}{\partial _x}{d_2} + {\partial _t}{b_1} + {\partial _u}{d_1} = 0\tag{iv}\label{de4}\end{equation} \begin{equation}{\partial _x}{b_1} - {\partial _t}T + {\partial _u}{c_1} - {d_2} = 0\tag{v}\label{de5}\end{equation} and we have \begin{equation}T = T\left( t \right)\notag\end{equation} \begin{equation}X = {b_1}\left( {x,t} \right)\notag\end{equation} \begin{equation}U = {c_1}\left( {x,t,u} \right)\notag\end{equation} \begin{equation}W = {d_1}\left( {x,t,u} \right) + w{d_2}\left( {x,t} \right)\notag\end{equation} Note \eqref{de5} contains only dependencies on $\left( {x,t} \right)$ except for the third term, requiring ${c_1}$ to have this functional form: \begin{equation}{c_1}\left( {x,t,u} \right) = {c_{11}}\left( {x,t} \right) + u{c_{12}}\left( {x,t} \right)\notag\end{equation} giving \begin{equation}{\partial _x}{b_1} - {\partial _t}T + {c_{12}} - {d_2} = 0\tag{\ref{de5}}\end{equation} A similar reasoning applies to \eqref{de4}, enabling: \begin{equation}{d_1}\left( {x,t,u} \right) = {d_{11}}\left( {x,t} \right) + u{d_{12}}\left( {x,t} \right)\notag\end{equation} giving \begin{equation}{\partial _x}{d_2} + {\partial _t}{b_1} + {d_{12}} = 0\tag{\ref{de4}}\end{equation} Insert our new form for ${c_1}$ and ${d_1}$ into our DEs: \begin{equation}{d_{11}}\left( {x,t} \right) - u{d_{12}}\left( {x,t} \right) + {\partial _x}{c_{11}} + u{\partial _x}{c_{12}} = 0\tag{\ref{de1}}\end{equation} \begin{equation} - {d_2} - \left( {{\partial _x}{b_1} - {c_{12}}} \right) = 0\tag{\ref{de2}}\end{equation} \begin{equation} - {\partial _t}{c_{11}} - u{\partial _t}{c_{12}} + {\partial _x}{d_{11}} + u{\partial _x}{d_{12}} = 0\tag{\ref{de3}}\end{equation} \begin{equation}{\partial _x}{d_2} + {\partial _t}{b_1} + {d_{12}} = 0\tag{\ref{de4}}\end{equation} \begin{equation}{\partial _x}{b_1} - {\partial _t}T + {\partial _u}{c_1} - {d_2} = 0\tag{\ref{de5}}\end{equation} Operate on \eqref{de1} with ${\partial _x}$: \begin{equation} - {\partial _x}{d_{11}}\left( {x,t} \right) - u{\partial _x}{d_{12}}\left( {x,t} \right) + {\partial _{xx}}{c_{11}} + u{\partial _{xx}}{c_{12}} = 0\notag\end{equation} Add this to \eqref{de3}: \begin{equation} - {\partial _x}{d_{11}}\left( {x,t} \right) - u{\partial _x}{d_{12}}\left( {x,t} \right) + {\partial _{xx}}{c_{11}} + u{\partial _{xx}}{c_{12}} - {\partial _t}{c_{11}} - u{\partial _t}{c_{12}} + {\partial _x}{d_{11}} + u{\partial _x}{d_{12}} = 0\notag\end{equation} yielding: \begin{equation} - {\partial _t}{c_{11}} + {\partial _{xx}}{c_{11}} - u\left( {{\partial _t}{c_{12}} - {\partial _{xx}}{c_{12}}} \right) = 0\notag\end{equation} We may conclude that individually ${c_{11}}$ and ${c_{12}}$ obey the heat equation (but ${c_1}\left( {x,t,u} \right)$ as a whole does not necessarily). Notice in operating on \eqref{de1} with ${\partial _{\text{x}}}$ we do not include a term ${u_x}{\partial _x}{c_{12}}$ since in jet space u and x are treated independent of each other, unlike in the Olver style prolongation process, where there is ambiguity.  Rewrite U now making explicit the functional forms: \begin{equation}T = T\left( t \right)\notag\end{equation} \begin{equation}X = {b_1}\left( {x,t} \right)\notag\end{equation} \begin{equation}U = {g_H}\left( {x,t} \right) + u{h_H}\left( {x,t} \right)\notag\end{equation} \begin{equation}W = {d_{11}}\left( {x,t} \right) + u{d_{12}}\left( {x,t} \right) + w{d_2}\left( {x,t} \right)\notag\end{equation} in which the subscripts H remind us they obey the heat equation. Our determining equations now look like: \begin{equation} - {d_{11}} - u{d_{12}} + {\partial _x}{g_H} + u{\partial _x}{h_H} = 0\tag{\ref{de1}}\end{equation} \begin{equation} - {d_2} - \left( {{\partial _x}{b_1} - {h_H}} \right) = 0\tag{\ref{de2}}\end{equation} \begin{equation} - {\partial _t}{g_H} - u{\partial _t}{h_H} + {\partial _x}{d_{11}} + u{\partial _x}{d_{12}} = 0\tag{\ref{de3}}\end{equation} \begin{equation}{\partial _x}{d_2} + {\partial _t}{b_1} + {{\text{d}}_{12}} = 0\tag{\ref{de5}}\end{equation} \begin{equation}{\partial _x}{b_1} - {\partial _t}T + {h_H} - {d_2} = 0\tag{\ref{de5}}\end{equation}

Notice the in \eqref{de1} and \eqref{de3} that contain factors of u must independently vanish since the other terms are explicitly independent of $u$.  So \eqref{de1} breaks into two equations:

\begin{equation} - {d_{11}} + {\partial _x}{g_H} = 0 \to {d_{11}} = {\partial _x}{g_H}\notag\end{equation} \begin{equation} - {d_{12}} + {\partial _x}{h_H} = 0 \to {d_{12}} = {\partial _x}{h_H}\notag\end{equation} as does \eqref{de3}: \begin{equation} - {\partial _t}{g_H} + {\partial _x}{d_{11}} = 0 \to {\partial _x}{d_{11}} = {\partial _t}{g_H}\notag\end{equation} \begin{equation} - {\partial _t}{h_H} + {\partial _x}{d_{12}} = 0 \to {\partial _x}{d_{12}} = {\partial _t}{h_H}\notag\end{equation}

These four relations are consistent with the definitions of ${g_H}$ and ${h_H}$.  For example, take ${\partial _x}$of the first and combine with the third: \begin{equation}{\partial _x}{d_{11}} = {\partial _{xx}}{g_H} = {\partial _t}{g_H}\notag\end{equation} Recapping \begin{equation}T = T\left( t \right)\notag\end{equation} \begin{equation}X = {b_1}\left( {x,t} \right)\notag\end{equation} \begin{equation}U = {g_H}\left( {x,t} \right) + u{h_H}\left( {x,t} \right)\notag\end{equation} \begin{equation}W = {\partial _x}{g_H} + u{\partial _x}{h_H} + w{d_2}\left( {x,t} \right)\notag\end{equation} So we still have untapped these determining equations: \begin{equation} - {d_2} - \left( {{\partial _x}{b_1} - {h_H}} \right) = 0\tag{\ref{de2}}\end{equation} \begin{equation}{\partial _x}{d_2} + {\partial_t}{b_1} + {\partial_x}{h_H} = 0\tag{\ref{de4}}\end{equation} \begin{equation}{\partial _x}{b_1} - {\partial_t}T + {h_H} - {d_2} = 0\tag{\ref{de5}}\end{equation} Look at some combinations.  Take ${\partial_x}$ of \eqref{de2} and use it to eliminate ${\partial_x}{h_H}$ by combining with the updated \eqref{de4}: \begin{equation}2{\partial _{\text{x}}}{d_2} + {\partial _{xx}}{b_1} + {\partial_t}{b_1} = 0\notag \end{equation} Now take $\partial_x$ of \eqref{de5} and use it to eliminate ${\partial_x}{h_H}$ in \eqref{de4}: \begin{equation}2{\partial _x}{d_2} + {\partial _t}{b_1} - {\partial _{xx}}{b_1} = 0\notag\end{equation} Subtracting these two equations yields: \begin{equation}{\partial _{xx}}{b_1} = 0\notag\end{equation} so \begin{equation}{b_1} = xf\left( t \right) + g\left( t \right)\notag\end{equation} Finally, let us add \eqref{de5} to \eqref{de2} \begin{equation} - 2{\partial _x}{b_1} + {\partial _t}T = 0 \to {\partial _t}T = 2{\partial _x}{b_1} = 2f\left( t \right)\notag\end{equation} Recapping \begin{equation}T = T\left( t \right)\notag\end{equation} \begin{equation}X = \frac{1}{2}x{\partial _t}T + g\left( t \right)\notag\end{equation} \begin{equation}U = {g_H}\left( {x,t} \right) + u{h_H}\left( {x,t} \right)\notag\end{equation} \begin{equation}W = {\partial _x}{g_H} + u{\partial _x}{h_H} + w{d_2}\left( {x,t} \right)\notag\end{equation} Our “untapped” three equations from above now look like \begin{equation} - {d_2} - f + {h_H} = 0\tag{\ref{de2}}\end{equation} \begin{equation}{\partial _x}{d_2} + x{\partial _t}f + {\partial _t}g\left( t \right) + {\partial _x}{h_H} = 0\tag{\ref{de4}}\end{equation} \begin{equation} - {\partial _t}T + {h_H} - {d_2} = 0\tag{\ref{de5}}\end{equation} Combining \eqref{de2} and \eqref{de5} we find \begin{equation} {\partial _t}T=2f\notag\end{equation} Then we can finally write \begin{equation}T = T\left( t \right)\notag\end{equation} \begin{equation}X = xf\left( t \right) + g\left( t \right);\;f\left( t \right) = \frac{1}{2}{\partial _t}T\notag\end{equation} \begin{equation}U = {g_H}\left( {x,t} \right) + u{h_H}\left( {x,t} \right)\notag\end{equation} \begin{equation}W = {\partial _x}{g_H} + u{\partial _x}{h_H} + w\left( {{h_H} - f\left( t \right)} \right)\notag\end{equation}

To make further progress we look for self-consistent solutions to our DEs using power series expansions.  Let: \begin{equation}T = {a_i}{t^i}\notag\end{equation} \begin{equation}f\left( t \right) = b_j^1{t^j}\notag\end{equation} \begin{equation}g\left( t \right) = b_k^2{t^k}\notag\end{equation} \begin{equation}{h_H} = {h_{jk}}{x^j}{t^k}\notag\end{equation}

where we let the indices run from $0 \to 3$, unless we find we need higher powers of $x\;{\text{and}}\;t$. Let ${g_H}$ ride for now, as it does not appear in the DEs. Let us employ these expansions in an equation formed by combining \eqref{de4} and \eqref{de5}, eliminating $d_2$ and eliminating/combining surviving terms: \begin{equation}x{\partial _t}f + {\partial _t}g + 2{\partial _x}{h_H} = 0\notag\end{equation} Now insert the expansions: \begin{equation}2{\partial _x}{h_{jk}}{x^j}{t^k} + x{\partial _t}b_j^1{t^j} + {\partial _t}b_k^2{t^k} = 0\notag\end{equation}

\begin{equation}2j{h_{jk}}{x^{j - 1}}{t^k} + xjb_j^1{t^{j - 1}} + kb_k^2{t^{k - 1}} = 0\notag\end{equation} Partially expanding: \begin{equation}0 + 2{h_{1k}}{x^0}{t^k} + 4{h_{2k}}{x^1}{t^k} + 6{h_{3k}}{x^2}{t^k} + 0 + x1b_1^1{t^0} + x2b_2^1{t^1} + x3b_3^1{t^2} + 0 + 1b_1^2{t^0} + 2b_2^2{t^1} + 3b_3^2{t^2} = 0{\text{\;\;}}\notag\end{equation} \begin{equation}2{h_{1k}}{x^0}{t^k} + 4{h_{2k}}{x^1}{t^k} + 6{h_{3k}}{x^2}{t^k} + x1b_1^1{t^0} + x2b_2^1{t^1} + x3b_3^1{t^2} + 1b_1^2{t^0} + 2b_2^2{t^1} + 3b_3^2{t^2} = 0{\text{\;\;}}\notag\end{equation} Let's work on the $t$-variable first.  Equate powers of $t$:

${t^0}$:

\begin{equation}2{h_{10}}{x^0}{t^0} + 4{h_{20}}{x^1}{t^0} + 6{h_{30}}{x^2}{t^0} + x1b_1^1{t^0} + 1b_1^2{t^0} = 0\notag\end{equation} \begin{equation}2{h_{10}} + 4{h_{20}}{x^1} + 6{h_{30}}{x^2} + xb_1^1 + b_1^2 = 0\notag\end{equation}

Employing monomials in $x$:

\begin{equation}2{h_{10}} + b_1^2 = 0\notag\end{equation} \begin{equation}4{h_{20}} + b_1^1 = 0\notag\end{equation} \begin{equation}6{h_{30}} = 0\notag\end{equation}
