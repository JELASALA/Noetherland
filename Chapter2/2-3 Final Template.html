2.3 – Flow Induced by a Vector Field:  Integral Curves

With this concrete, though elementary introduction via examples, let's move to Olver's more formal approach, needed for the main result. We imagine a generic parameterized curve \begin{equation}\phi \left( \varepsilon  \right) = \left( {{\phi ^1}\left( \varepsilon  \right),{\phi ^2}\left( \varepsilon  \right), \ldots ,{\phi ^m}\left( \varepsilon  \right)} \right)\notag\end{equation} in $m$ dimensions ($\varepsilon $ is not infinitesimal at this point).  Now compared to the introductory examples we have the replacements: \begin{equation}\gamma  \to \phi \notag\end{equation} \begin{equation}s \to \varepsilon \notag\end{equation} and \begin{equation}{x^i}\left( s \right) \to {\phi ^i}\left( \varepsilon  \right)\notag\end{equation} to be consistent with Olver, so the ${\phi ^i}$ are independent functions of the parameter $\varepsilon $. Thus our example from earlier: \begin{equation}\gamma \left( \theta  \right) = \left( {x\left( \theta  \right),y\left( \theta  \right)} \right) = \left( {r\cos \theta ,r\sin \theta } \right)\notag\end{equation} now looks like \begin{equation}\phi \left( \varepsilon  \right) = \left( {{\phi ^1}\left( \varepsilon  \right),{\phi ^2}\left( \varepsilon  \right)} \right) = \left( {r\cos \varepsilon ,r\sin \varepsilon } \right)\notag\end{equation} We now use $x \equiv \left\{ {{x^i}} \right\}$ to represent the $m$ local coordinates, and we focus on the specific point $x$ that the curve passes through.   At that point the curve has a tangent vector $\dot \phi \left( \varepsilon  \right)$, where the dot indicates the derivative wrt $\varepsilon $.  With a coordinate system defined, Olver often uses ${x^i}\left( \varepsilon  \right)$ (instead of the ${\phi ^i}\left( \varepsilon  \right)$) to represent the (components of the) curve (aka trajectory).  Next he identifies the object \begin{equation}v{|_x} = {\xi ^1}\left( x \right){\partial _{{x^1}}} + {\xi ^2}\left( x \right){\partial _{{x^2}}} +  \ldots  + {\xi ^m}\left( x \right){\partial _{{x^m}}} \equiv \xi \left( x \right){\partial _x}\label{231}\end{equation} called the vector field at point x, as the tangent vector of $\phi $.  Symbolically he writes this relation two ways, one (unlabeled) using $\phi $ (most generic, coordinate free) and another in local coordinates: \begin{equation}\dot \phi \left( \varepsilon  \right) = v{|_{\phi \left( \varepsilon  \right)}}\label{232}\end{equation} \begin{equation}\frac{{d{x^i}\left( \varepsilon  \right)}}{{d\varepsilon }}{|_{\varepsilon  = 0}} = {\xi ^i}\left( x \right)\label{O17}\tag{Olver (1.7)}\end{equation} Relation \eqref{232} implies $\phi $ is the integral curve or flow of the vector $v$ at all points $\varepsilon $ along its trajectory.  That is, $v$ generates the curve $\phi $ with a process we will see below (this was demonstrated earlier with the rotation curve).  So far Olver is using the subscripts ${|_x}$ and ${|_{\phi \left( \varepsilon  \right)}}$ to indicate the equality applies either at some specific point (in a coordinate system) or at a generic point on the trajectory, respectively. Note \eqref{O17} will turn out to be most useful in determining curve parameterization for various forms of the vector field functions ${\xi ^i}\left( x \right)$.  Note that we have inserted the condition ${|_{\varepsilon  = 0}}$ on the LHS which Olver leaves out.  We do this because the RHS is to be understood at a specific point in the coordinate system, where we presume $\varepsilon  = 0$ applies.  Also we do not find it logical to have a "free" $\varepsilon $ on the LHS without one on the RHS of the equation.

Next Olver grants $\phi $ a special symbol when it is the maximal integral curve.  We don't think this is really very important, but he makes the symbolic transition   \begin{equation}\phi \left( \varepsilon  \right) \to \Psi \left( {\varepsilon ,x} \right)\notag\end{equation} which is really still another name for $x\left( \varepsilon  \right)$.  Olver points out that $\Psi \left( {0,x} \right) = x$.  Then   \begin{equation}\dot \Psi \left( {\varepsilon ,x} \right) = \frac{d}{{d\varepsilon }}\Psi \left( {\varepsilon ,x} \right) = v{|_{\Psi \left( {\varepsilon ,x} \right)}}\label{O111}\tag{Olver (1.11)}\end{equation} which he says means "$v$ is the tangent vector to the curve $\Psi \left( {\varepsilon ,x} \right)$ for fixed $x$." The curve has the (unsurprising) property \begin{equation}\Psi \left( {\delta ,\Psi \left( {\varepsilon ,x} \right)} \right) = \Psi \left( {\delta  + \varepsilon ,x} \right)\notag\end{equation} which in words says we are simply inching our way along the curve's parameter values to explore further curve values along this maximal path.  Olver calls $\Psi $ the flow generated by $v$, defined in Chapter 1 as the "parameterized maximal integral curve passing through $x$ in $M$." The flow (curve) and the vector field that generates it are connected by this remarkable relation (not even given a number by Olver): \begin{equation}{\text{exp}}\left( {\varepsilon v} \right)x \equiv \Psi \left( {\varepsilon ,x} \right)\notag\end{equation} where we note Olver employs a rarely used definition symbol.  This is consistent with the properties of the flow under the group-like behaviors: \begin{equation}\Psi \left( {\delta ,\Psi \left( {\varepsilon ,x} \right)} \right) = {\text{exp}}\left( {\delta v} \right)\Psi \left( {\varepsilon ,x} \right) = {\text{exp}}\left( {\delta v} \right){\text{exp}}\left( {\varepsilon v} \right)x = {\text{exp}}\left( {\left( {\delta  + \varepsilon } \right)v} \right)x = \Psi \left( {\delta  + \varepsilon ,x} \right)\notag\end{equation} \begin{equation}\Psi \left( {0,x} \right) = {\text{exp}}\left( {0v} \right)x = x\notag\end{equation} and \begin{equation}\dot \Psi \left( {\varepsilon ,x} \right) = \frac{d}{{d\varepsilon }}\Psi \left( {\varepsilon ,x} \right) = \frac{d}{{d\varepsilon }}\left[ {{\text{exp}}\left( {\varepsilon v} \right)x} \right] = v\left[ {{\text{exp}}\left( {\varepsilon v} \right)x} \right] \equiv v{|_{\Psi \left( {\varepsilon ,x} \right)}}\tag{Olver (1.11')}\label{O111'}\end{equation} That is, the derivative of the "flow" or "curve" or "trajectory" along its parameter is equivalent to the action of the generating vector on that curve.  This can be validated easily and will prove useful.  

Aside

Confusion may accompany the meaning of the symbol ${|_{\Psi \left( {\varepsilon ,x} \right)}}$.  In the step before the final expression above we see $v$ actually operates on the subscripted variable, a conclusion validated in the examples below.  Looking back, say at \eqref{231} \begin{equation}v{|_x} = {\xi ^1}\left( x \right){\partial _{{x^1}}} + {\xi ^2}\left( x \right){\partial _{{x^2}}} +  \ldots  + {\xi ^m}\left( x \right){\partial _{{x^m}}} \equiv \xi \left( x \right){\partial _x}\notag\end{equation} we must scratch our heads. However, if one did interpret the subscript as being operated on, then \begin{equation}v{|_{{x^j}}} = {\xi ^i}\left( x \right){\partial _{{x^i}}}{x^j} = {\xi ^i}\left( x \right)\delta _i^j = {\xi ^j}\left( x \right)\notag\end{equation} meaning it "plucks out" the indicated coefficient without its associated basis vector.  Olver likely meant for this to mean "at the point $x$" rather than operating to generate the coefficients, but that cannot be the correct interpretation of its use in \eqref{O111} since operation is required.Indeed, at the far left we start with a function, so must end with a function (not an operator) at the far right.  So we conclude that proper interpretation must be the student's burden.

EXAMPLES:

1.  Take $\xi \left( x \right) = 1 \to v = {\partial _x}$, so \begin{equation}\Psi \left( {\varepsilon ,x} \right) = {\text{exp}}\left( {\varepsilon v} \right)x = \left( {1 + \varepsilon v + \frac{1}{2}{{\left( {\varepsilon v} \right)}^2} +  \ldots } \right)x = x + \varepsilon {\partial _x}x + \frac{1}{2}{\varepsilon ^2}{\partial _x}{\partial _x}x +  \ldots  = x + \varepsilon \notag\end{equation} which Olver points out is globally defined (i.e., no singularities through the range of the coordinate system).  Note there is no requirement for $\varepsilon $ to be infinitesimal.  We may call this a simple "displacement" transformation.  That is, such a vector produces a linear trajectory in $\varepsilon $ along the x-axis.  Let's check the interpretation of \eqref{O111'}: \begin{equation}\dot \Psi \left( {\varepsilon ,x} \right) = 1 = ? = v{|_{\Psi \left( {\varepsilon ,x} \right)}} = {\partial _x}\left( {x + \varepsilon } \right) = 1 = \xi \;\;\emptyset \notag\end{equation} 2.  Take $\xi \left( x \right) = x \to v = x{\partial _x}$, so \begin{equation}\Psi \left( {\varepsilon ,x} \right) = {\text{exp}}\left( {\varepsilon v} \right)x = \left( {1 + \varepsilon v + \frac{1}{2}{{\left( {\varepsilon v} \right)}^2} +  \ldots } \right)x\notag\end{equation} \begin{equation} = x + \varepsilon x{\partial _x}x + \frac{1}{2}{\varepsilon ^2}x{\partial _x}\left( {x{\partial _x}} \right)x = x + \varepsilon x + \frac{1}{2}{\varepsilon ^2}x +  \ldots  = \;x{e^\varepsilon }\notag\end{equation} This we would call "dilation" or "scaling."  That is, this vector magnifies the value of x exponentially with increasing $\varepsilon $.  Olver points out that this result can be seen from the solution to the ODE $\dot x = x$.  Where does this come from?  Perhaps he is referring to \eqref{O111} above which might be interpreted as \begin{equation}\frac{{d\Psi \left( {\varepsilon ,x} \right)}}{{d\varepsilon }} = \xi \left( x \right) = x\notag\end{equation} Replacing $\Psi \left( {\varepsilon ,x} \right)\;{\text{with}}\;x\left( \varepsilon  \right)$ gives the equation $\dot x = x$.  Let's see what we get by solving this: \begin{equation}\frac{{dx\left( \varepsilon  \right)}}{{d\varepsilon }} = x\left( \varepsilon  \right) \to \frac{{dx\left( \varepsilon  \right)}}{{x\left( \varepsilon  \right)}} = d\varepsilon  \to \ln \left( x \right)|_0^\varepsilon  = \varepsilon  \to \ln \left( {x\left( \varepsilon  \right)/x\left( 0 \right)} \right) = \varepsilon  \to {e^\varepsilon } = \frac{{x\left( \varepsilon  \right)}}{{x\left( 0 \right)}} \to x\left( \varepsilon  \right) = x\left( 0 \right){e^\varepsilon }\notag\end{equation} If we take $x\left( 0 \right)$ to be represented by the "initial" point $x$, then we have $x\left( \varepsilon  \right) = x{e^\varepsilon } \equiv \Psi \left( {\varepsilon ,x} \right)$. We can also test \eqref{O111}: \begin{equation}\dot \Psi \left( {\varepsilon ,x} \right) = x{e^\varepsilon } = ? = v{|_{\Psi \left( {\varepsilon ,x} \right)}} = x{\partial _x}x{e^\varepsilon } = x{e^\varepsilon }\notag\end{equation} In the limit $\varepsilon  \to $ we have \begin{equation}\dot \Psi \left( {0,x} \right) = x = \xi \;\color{blue}\blacksquare \notag\end{equation}

Proceeding: \begin{equation}v = {x^2}{\partial _x} \to \Psi \left( {\varepsilon ,x} \right) = {\text{exp}}\left( {\varepsilon v} \right)x = \left( {1 + \varepsilon v + \frac{1}{2}{{\left( {\varepsilon v} \right)}^2} +  \ldots } \right)x\notag\end{equation} \begin{equation} = x + \varepsilon {x^2}{\partial _x}x + \frac{1}{2}{\varepsilon ^2}{x^2}{\partial _x}\left( {{x^2}{\partial _x}} \right)x +  \ldots  = x + \varepsilon {x^2} + \frac{1}{2}{\varepsilon ^2}2{x^3} +  \ldots \notag\end{equation} \begin{equation} = x\left( {1 + \varepsilon x + {\varepsilon ^2}{x^2} +  \ldots } \right) = \frac{x}{{1 - \varepsilon x}}\notag\end{equation} Clearly this is not globally defined.  This "curve" starts at x and grows even more strongly with $\varepsilon $ than the previous example.  Can we apply the logic used earlier to "derive" this result more simply? \begin{equation}\frac{{d{\text{x}}\left( \varepsilon  \right)}}{{d\varepsilon }} = {x^2}\left( \varepsilon  \right) \to \frac{{d{\text{x}}\left( \varepsilon  \right)}}{{{x^2}\left( \varepsilon  \right)}} = d\varepsilon  \to  - \frac{1}{x}|_0^\varepsilon  = \varepsilon  \to  - \frac{1}{{{\text{x}}\left( \varepsilon  \right)}} + \frac{1}{{{\text{x}}\left( 0 \right)}} = \varepsilon \notag\end{equation} Replacing ${\text{x}}\left( 0 \right)$ with x and solving for ${\text{x}}\left( \varepsilon  \right)$ as before: \begin{equation}\frac{1}{{{\text{x}}\left( \varepsilon  \right)}} = \frac{1}{{\text{x}}} - \varepsilon  \to {\text{x}}\left( \varepsilon  \right) = \frac{x}{{1 - \varepsilon x}} = \Psi \left( {\varepsilon ,x} \right)\;\emptyset \notag\end{equation} We can also test: \begin{equation}\dot \Psi \left( {\varepsilon ,x} \right) = \frac{{{x^2}}}{{{{\left( {1 - \varepsilon x} \right)}^2}}} = ? = v{|_{\Psi \left( {\varepsilon ,x} \right)}} = {x^2}{\partial _x}\left( {\frac{x}{{1 - \varepsilon x}}} \right) = {x^2}{\partial _x}\left( {\frac{1}{{1 - \varepsilon x}} + \frac{{x\varepsilon }}{{{{\left( {1 - \varepsilon x} \right)}^2}}}} \right) = \frac{{{x^2}}}{{{{\left( {1 - \varepsilon x} \right)}^2}}}\notag\end{equation} In the limit $\varepsilon \to $ we have \begin{equation}\dot \Psi \left( {0,x} \right) = {x^2} = \xi \;\color{blue}\blacksquare \notag\end{equation}




Proceeding: \begin{equation}v =  - y{\partial _x} + x{\partial _y} \to \Psi \left( {\varepsilon ,\left( {x,y} \right)} \right) = {\text{exp}}\left( {\varepsilon v} \right)\left( {x,y} \right) = \left( {1 + \varepsilon v + \frac{1}{2}{{\left( {\varepsilon v} \right)}^2} + \frac{1}{{3!}}{{\left( {\varepsilon v} \right)}^3} +  \ldots } \right)\left( {x,y} \right)\notag\end{equation} \begin{equation} = \left( {1 + \varepsilon \left( { - y{\partial _x} + x{\partial _y}} \right) + \frac{1}{2}{\varepsilon ^2}{{\left( { - y{\partial _x} + x{\partial _y}} \right)}^2} + \frac{1}{{3!}}{\varepsilon ^3}{{\left( { - y{\partial _x} + x{\partial _y}} \right)}^3} +  \ldots } \right)\left( {x,y} \right)\notag\end{equation} \begin{equation} = \left( {\left( {x,y} \right) + \varepsilon \left( { - y,x} \right) + \frac{1}{2}{\varepsilon ^2}\left( { - y{\partial _x} + x{\partial _y}} \right)\left( { - y,x} \right) + \frac{1}{{3!}}{\varepsilon ^3}\left( { - y{\partial _x} + x{\partial _y}} \right)\left( { - y{\partial _x} + x{\partial _y}} \right)\left( { - y,x} \right) +  \ldots } \right)\notag\end{equation} \begin{equation} = \left( {\left( {x,y} \right) + \varepsilon \left( { - y,x} \right) + \frac{1}{2}{\varepsilon ^2}\left( { - x, - y} \right) + \frac{1}{{3!}}{\varepsilon ^3}\left( { - y{\partial _x} + x{\partial _y}} \right)\left( { - x, - y} \right) +  \ldots } \right)\notag\end{equation} \begin{equation} = \left( {\left( {x,y} \right) + \varepsilon \left( { - y,x} \right) + \frac{1}{2}{\varepsilon ^2}\left( { - x, - y} \right) + \frac{1}{{3!}}{\varepsilon ^3}\left( {y, - x} \right) +  \ldots } \right)\notag\end{equation} With a little foresight we can rearrange this as: \begin{equation}{\Psi ^1}\left( {\varepsilon ,\left( {x,y} \right)} \right) = x\left( {1 - \frac{1}{2}{\varepsilon ^2} +  \ldots } \right) - y\left( {\varepsilon  - \frac{1}{{3!}}{\varepsilon ^3} +  \ldots } \right) = x\cos \varepsilon  - y\sin \varepsilon \notag\end{equation} \begin{equation}{\Psi ^2}\left( {\varepsilon ,\left( {x,y} \right)} \right) = y\left( {1 - \frac{1}{2}{\varepsilon ^2} +  \ldots } \right) + x\left( {\varepsilon  - \frac{1}{{3!}}{\varepsilon ^3} +  \ldots } \right) = y\cos \varepsilon  + x\sin \varepsilon \notag\end{equation} So the flow (curve) $\Psi \left( {\varepsilon ,x} \right)$ generated by this vector $v$ is given by \begin{equation}\Psi \left( {\varepsilon ,x} \right) = \left( {{\Psi ^1}\left( {\varepsilon ,\left( {x,y} \right)} \right),{\Psi ^2}\left( {\varepsilon ,\left( {x,y} \right)} \right)} \right) = \left( {x\cos \varepsilon  - y\sin \varepsilon ,y\cos \varepsilon  + x\sin \varepsilon } \right)\notag\end{equation} This could also have been derived far easier solving together \begin{equation}\frac{{dx\left( \varepsilon  \right)}}{{d\varepsilon }} =  - Example 4 produced the integral curve \begin{equation}\left( {{\Psi ^1},{\Psi ^2}} \right) = \left( { - y\sin \varepsilon  + x\cos \varepsilon ,y\cos \varepsilon  + x\sin \varepsilon } \right)\notag\end{equation} So what does this actually mean?  This is best seen graphically.  Let's select initial points $x = 1$ and $y = 2$ and then propagate them IAW the flow.


If we imagine any given point on the integral curve, or flow, as a vector pointing from the origin, we can ask about the magnitude of that vector from the scalar product: \begin{equation}\Psi \left( {\varepsilon ,x} \right)\cdot \Psi \left( {\varepsilon ,x} \right) = {x^2} + {y^2} = 5 \to \text{radius} = \sqrt 5 = 2.23\notag\end{equation} consistent with the figure. Finally we can test again interpretation of \eqref{O111'}: \begin{equation}\dot \Psi \left( {\varepsilon ,\left( {x,y} \right)} \right) = \left( { - x\sin \varepsilon  - y\cos \varepsilon , - y\sin \varepsilon  + x\cos \varepsilon } \right) = ? = v{|_{\Psi \left( {\varepsilon ,x} \right)}}\notag\end{equation} \begin{equation} = \left( { - y{\partial _x} + x{\partial _y}} \right)\left( {x\cos \varepsilon  - y\sin \varepsilon ,y\cos \varepsilon  + x\sin \varepsilon } \right)\notag\end{equation} \begin{equation} = \left( { - y\cos \varepsilon  - x\sin \varepsilon ,x\cos \varepsilon  - y\sin \varepsilon } \right)\notag\end{equation} In the limit $\varepsilon  \to $ we have \begin{equation}\dot \Psi \left( {0,x} \right) = \left( { - y,x} \right) = \left( {{\xi ^1},{\xi ^2}} \right)\;\color{blue}\blacksquare\notag\end{equation} In this section we demonstrated the process for generating the integral curve, or flow, of an infinitesimal vector generator.
