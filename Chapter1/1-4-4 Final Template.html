Section 1.4.4 LLA to the Function-Field

We are going to examine the following function in ${\mathbb{R}^2}$

\begin{equation}f\left( {x,y} \right) = {x^2} + {y^2}\notag\end{equation}

and find its local linear approximation near the point $P:\left( {{x_o},{y_o}} \right)$.  In this example we will still use $\left( {{x_o},{y_o}} \right) = \left( {1,1} \right)$ as that is the same point considered in the curve map we just finished.  The circle that goes through $\left( {{x_o},{y_o}} \right) = \left( {1,1} \right)$ is a circle with its center at the origin and with a radius given by \begin{equation}r = \sqrt {\left[ {{{\left( {{x_o}} \right)}^2} + {{\left( {{y_o}} \right)}^2}} \right]}  = \sqrt 2 \notag\end{equation} and the function a value of 2.

One of the fundamental differences between $\gamma \left( s \right)$, a curve map, and $f\left( {x,y} \right)$, a function map, is that $f\left( {x,y} \right)$ is defined over a "region" of M whereas $\gamma \left( s \right)$ is defined only along a continuous set of points in M, known as "the curve".  We need some way to visualize f's local, linear behavior over a region, and in the limit in the vicinity of a point.  We may represent a function-field graphically with a series of contour lines at fixed “heights.”  Here is an example of the elevation map of the island of Hawaii.  Typically contour lines are shown for fixed valued of elevation, here shown in feet above sea level.


Our ongoing example function

\begin{equation}f\left( {x,y} \right) = {x^2} + {y^2}\notag\end{equation}

when set to a real number constant will give us a contour line.  A reasonable way to represent the field associated with our function in terms of contour lines is to set the function equal to integers.  Each integer will produce a circle.  Note the circles will not be equally spaced in the manifold.  We set

\begin{equation}f\left( {x,y} \right) = {x^2} + {y^2} = n\notag\end{equation}

with $n$  an integer and then a moment's thought reveals that $n$ is the square of the radius $r$ of the circular contour line and the square root of $n$ is the radius $r$ .  So going from $n$  to $n + 1$ is not going to produce a sequence of radii that is equally spaced.  This motivates zooming in on the local behavior of the function where this function (and any well behaved function) will behave linearly.  This is facilitated by our magnified/expanded coordinates utilized in the last section.  Define

\begin{equation}x = {x_o} + \varepsilon {X^*}\;\;\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;\;y = {y_o} + \varepsilon {Y^*}\;\;\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;\;n = {n_o} + \varepsilon N\notag\end{equation}

Notice that we've performed the analogous infinitesimal approximation to the parameter $n$ that we did for the parameter $s$.  We have also distinguished the magnified coordinates used here with an asterisk.  And just like $S$ became a rate in a new space attached at the remaining manifold point, $N$ will become a rate/variable of contours in a new space attached at the same manifold point.  In the manifold, the contours are given by

\begin{equation}f\left( {x,y} \right) = {x^2} + {y^2} = n\notag\end{equation}

Substituting our infinitesimal approximations, we find the contours given by

\begin{equation}{\left( {{x_o} + \varepsilon {X^*}} \right)^2} + {\left( {{y_o} + \varepsilon {Y^*}} \right)^2} = {n_o} + \varepsilon N\notag\end{equation}

Differentiating with respect to $\varepsilon $ we have

\begin{equation}2\left( {{x_o} + \varepsilon {X^*}} \right){X^*} + 2\left( {{y_o} + \varepsilon {Y^*}} \right){Y^*} = N\notag\end{equation}

We take the limit as $\varepsilon  \to 0$ to get to the tangent space and obtain the equation for the LLAs to our function

\begin{equation}2{x_o}{X^*} + 2{y_o}{Y^*} = N\notag\end{equation}

This demonstrates that regardless of the specific geometry of the base manifold, the geometry of the tangent space is Euclidean.  That equation in the magnified variables $\left( {X,Y} \right)$ is a Euclidean straight line, period.  There are no hidden curvature effects.  The tangent space is flat, to use a relativity term.

So let's solve that equation as a line:

\begin{equation}{Y^*} =  - \left( {\frac{{{x_o}}}{{{y_o}}}} \right){X^*} + \frac{N}{{2{y_o}}}\notag\end{equation}

At our common point of interest $\left( {{x_o},{y_o}} \right) = \left( {1,1} \right)$ we have:

\begin{equation}{Y^*} =  - {X^*} + \frac{N}{2}\notag\end{equation}
At the manifold coordinates $\left( {{x_o},{y_o}} \right) = \left( {1,1} \right)$ the slope of this line is -1, and as we vary N we vary the ${Y^*}$-intercept and produce an infinite set of parallel lines all of slope -1.  If we set N equal to successive integers we will create a set of equally spaced lines and that could represent the LLA in some sort of graphical way.  The$\Delta {Y^*} - {\text{intercept}}$s differ by a constant amount if we use integers for N:

\begin{equation}\Delta {Y^*} - {\text{intercept}} = \frac{{N + 1}}{2} - \frac{N}{2} = \frac{1}{2} = 0.5\notag\end{equation}
