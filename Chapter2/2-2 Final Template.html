2.2 – Infinitesimal Vector Generators

Why infinitesimal generators?  This is something discovered long ago by Sophus Lie and we will get there after discussing the idea of invariant functions.  A vector (field) acting at a point in a manifold might be represented as \begin{equation}\pmb{v} = {\xi ^i}\frac{\partial }{{\partial {x^i}}}\notag\end{equation} where the $\partial /\partial {x^i}$ are the basis vectors.  This might be the tangent vector to a curve, as suggested in the diagram below.


More generally, it might be a vector field representing a point-by-point transformation. For example, consider a rotation, which looks like this in two dimensions: \begin{equation}\pmb{v} =  - y\frac{\partial }{{\partial x}} + x\frac{\partial }{{\partial y}}\notag\end{equation} Consider the result of applying this vector to a function $f = f\left( {x,y} \right)$ and demanding the result vanish: \begin{equation}\pmb{v}\left( f \right) =  - y\frac{{\partial f}}{{\partial x}} + x\frac{{\partial f}}{{\partial y}} = 0\notag\end{equation} We say this function $f$ is an invariant of the vector field since the operation is required to vanish.  Let us solve for the function that satisfies this relationship.  We can use the MOC, explained in our section on Method of Characteristics [section number].  We use the non-parameterized form of MOC:
\begin{equation}\frac{{dx}}{{ - y}} = \frac{{dy}}{x}\notag\end{equation}
This solves easily:
\begin{equation}xdx =  - ydy \to {x^2} =  - {y^2} + C\notag\end{equation} We assign the function to the"constant" c, or better described "invariant", meaning $f$ satisfies the invariance condition when \begin{equation}f = {x^2} + {y^2} = C\left( {x,y} \right) \equiv {r^2}\left( {x,y} \right) = constant\notag\end{equation} Testing: \begin{equation}\pmb{v}\left( f \right) =  - y\frac{{\partial f}}{{\partial x}} + x\frac{{\partial f}}{{\partial y}} =  - 2yx + 2xy = 0\;\;\emptyset \notag\end{equation} Thus this rotation-inducing vector transformation defines invariant curves – circles in the x-y plane of radii $r$.  We can parameterize this curve (treating $r$ as a constant) as \begin{equation}\gamma \left( \theta  \right) = \left( {x\left( \theta  \right),y\left( \theta  \right)} \right) = \left( {r\cos \theta ,r\sin \theta } \right)\notag\end{equation} We can compute the tangent vector, or LLA, of this curve using, for example Burke’s notation (Burke p.__): [Tag LLA formula]
\begin{equation}\dot \gamma \left( \theta  \right) = \frac{{d{\gamma ^j}}}{{d\theta }}\frac{\partial }{{\partial {x^j}}}\notag\end{equation} So in this example: \begin{equation}\dot \gamma \left( \theta  \right) = \left( { - r\sin \theta ,r\cos \theta } \right) = \left( { - y,x} \right) =  - y\frac{\partial }{{\partial x}} + x\frac{\partial }{{\partial y}} = \pmb{v}\notag\end{equation} Main Point:  We see that a vector (field) acting on a function $f$ defines an "invariant" curve that satisfies $\pmb{v}\left( f \right) = 0$ and the tangent vector to the curve so defined is that generating vector.  Seems circular but it is, in fact, a nice tight self-consistent result. A simple additional example is given by \begin{equation}\pmb{v} = a\frac{\partial }{{\partial x}} + b\frac{\partial }{{\partial y}}\notag\end{equation} \begin{equation}\pmb{v}\left( f \right) = a\frac{{\partial f}}{{\partial x}} + b\frac{{\partial f}}{{\partial y}} = 0\notag\end{equation} MOC: \begin{equation}\frac{{dx}}{a} = \frac{{dy}}{b}\notag\end{equation}
This solves easily: \begin{equation}bdx = ady \to bx = ay - c\notag\end{equation} We assign the function to the "constant" $c$, or better described "invariant", meaning $f$ satisfies the invariance condition when \begin{equation}f = y - \frac{b}{a}x = c\notag\end{equation} Thus the vector generates straight lines with slope $b/a$ and various intercepts.  We can represent the lines as a curve (aka trajectory) parameterized by a real number $s$: \begin{equation}\gamma \left( s \right) = \left( {x\left( s \right),y\left( s \right)} \right) = \left( {as,bs + c} \right)\notag\end{equation} whose tangent vector takes us back to where we started: \begin{equation}\dot \gamma \left( s \right) = \left( {a,b} \right) = a\frac{\partial }{{\partial x}} + b\frac{\partial }{{\partial y}}\notag\end{equation}
