2.7 G-Invariant Solutions for the 2D Wave Equation

2.7.1 - Introduction

In Section 2.6 we found G-invariant solutions for the case of $p = 2,\;q = 1$, which were found using a single symmetry vector, constraining two of the variables on the solution manifold to generate an ODE that was directly solvable. That solution represents the behavior of the original EOM in the sub-manifold specified by the invariants of the invariance condition.

The purpose of this section is to use the Olver process to find invariant solutions for a PDE with three independent variables, requiring two “levels” of variable coupling via two different symmetry vectors. We will treat the well-studied 2D wave equation: \begin{equation}{u_{tt}} - {c^2}\left( {{u_{xx}} + {u_{yy}}} \right) = 0\notag\end{equation} or in units of $c = 1$: \begin{equation}{u_{tt}} - \left( {{u_{xx}} + {u_{yy}}} \right) = 0\notag\end{equation} which is an example of $p = 3,\;q = 1$. The complete set of vector generators of infinitesimal symmetries for the 2D wave equation are listed in Olver and at the end of Appendix 2A. We consider several of them pairwise, which reduces the PDE to an ODE.

We will discover that solutions are only possible when the two symmetry vectors employed are “compatible,” meaning they form a sub-group under commutation; that is, they either commute or their commutator is produces one of vectors. This is illustrated in Appendix 2B.

A note on notation: We have until now used a capital $V$ to indicate vectors, a notation that will extend into Chapter 3 as well. In this sub-section we will utilize Olver's notation for the various symmetry vectors for the 2DWE, lower case and boldfaced.

The vectors we will use include the following

Translations or “boosts” are defined when we set all constants to zero except ${c_1},\;{c_2},\;{\text{and}}/{\text{or}}\;{c_3}$ \begin{equation}{\partial _x}\;\;\;\;\;{\partial _y}\;\;\;\;{\partial _t}\;\notag\end{equation} which shift the origin of the independent variables in space/time: \begin{equation}{G_{1x}}:\;\;\;\left( {x,y,t,u} \right)\mapsto \left( {\bar x = x + \varepsilon ,\bar y = y,\bar t = t,\bar u = u} \right)\notag\end{equation} with analogous transformations for $y$ and $t$.

Scalings obtained by setting all constants to zero except ${c_4}:$ \begin{equation}\pmb{d} = x{\partial _x} + y{\partial _y} + t{\partial _t}\notag\end{equation} \begin{equation}{G_4}:\;\;\;\left( {x,y,t,u} \right)\mapsto \left( {\bar x = \lambda x,\bar y = \lambda y,\bar t = \lambda t,\bar u = u} \right)\notag\end{equation} There is a related vector seemingly plucked out of air by Olver but subsequently shown (by us in the Chapter 5 Noether’s Theorem) to be a consequence of RTI-like determining equations: \begin{equation}\pmb{m} \equiv \pmb{d} - \frac{1}{2}u{\partial _u} = x{\partial _x} + y{\partial _y} + t{\partial _t} - \frac{1}{2}u{\partial _u}\notag\end{equation} The last term can be seen to arise from \begin{equation}\phi \left( {x,y,t,u} \right) = \beta u + \alpha \notag\end{equation} \begin{equation}\beta = {c_{11}} - {c_8}x - {c_9}y - {c_{10}}t\notag\end{equation} with ${c_{11}} = - \frac{1}{2}{c_4}$.

Rotations (hyperbolic or Lorentz transformations for the latter two) are generated by taking ${c_5} \ne 0$, then ${c_6} \ne 0$, then ${c_7} \ne 0$ with the rest zero: \begin{equation}{\pmb{r}_{xy}} = - y{\partial _x} + x{\partial _y}\;\notag\end{equation} \begin{equation}{\pmb{r}_{xt}} = t{\partial _x} + \frac{x}{{{c^2}}}{\partial _t}\notag\end{equation} \begin{equation}{\pmb{r}_{yt}} = t{\partial _y} + \frac{y}{{{c^2}}}{\partial _t}\notag\end{equation} with corresponding transformations \begin{equation} \begin{split} {G_{5xy}}:\left( {x,y,t,u} \right)&\;\mapsto \;\left( {\bar x = x - \theta y,\bar y = \theta x + y,\bar t = t,\bar u = u} \right)\;\;\mapsto \;\left( {\bar x = \cos \theta x - \sin \theta y,\bar y = \sin \theta x + \cos \theta y,\bar t = t,\bar u = u} \right)\\ {G_{6xt}}:\left( {x,y,t,u} \right)&\;\mapsto \;\left( {\bar x = x + \theta t,\bar y = y,\bar t = \theta t + x,\bar u = u} \right)\;\;\mapsto \;\left( {\bar x = \cosh \theta x + \sinh \theta t,\bar y = y,\bar t = \sinh \theta t + \cosh \theta x,\bar u = u} \right) \end{split} \notag\end{equation}

In the next few sections we appy different pairs of symmetry vectors to reduce the 2D Wave Equation to an ODE.
