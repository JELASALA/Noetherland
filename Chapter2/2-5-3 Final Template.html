2.5.3 – Group Transformations for the 1D Heat Equation

The final step is generation of the group transforms, some samples of which we demonstrated in Examples above.  This employs the exponentiation process, as the vectors derived above are the infinitesimal generators of the symmetries.  For a two-component \begin{equation}\Psi \left( {\varepsilon ,\left( {x,t,u} \right)} \right) = {\text{exp}}\left( {\varepsilon \pmb{v}} \right)\left( {x,t,u} \right) = \left( {1 + \varepsilon \pmb{v} + \frac{1}{2}{{\left( {\varepsilon \pmb{v}} \right)}^2} +  \ldots } \right)\left( {x,t,u} \right)\notag\end{equation} Let's try \begin{equation}\pmb{v_5} = 2t{\partial _x} - xu{\partial _u}\notag\end{equation}
Do it by components \begin{equation}{\Psi ^x}\left( {\varepsilon ,\left( {x,t,u} \right)} \right) = {\text{exp}}\left( {\varepsilon \pmb{v_5}} \right)\left( x \right) = \left( {1 + \varepsilon \pmb{v_5} + \frac{1}{2}{{\left( {\varepsilon \pmb{v_5}} \right)}^2} +  \ldots } \right)\left( x \right) = x + \varepsilon \pmb{v_5}x + \frac{1}{2}{\varepsilon ^2}\pmb{v_5}\left( {\pmb{v_5}x} \right) +  \ldots \notag\end{equation} \begin{equation}{\Psi ^x}\left( {\varepsilon ,\left( {x,t,u} \right)} \right) = x + \varepsilon 2t + \frac{1}{2}{\varepsilon ^2}\pmb{v_5}\left( {2t} \right) +  \ldots  = x + 2\varepsilon t \equiv \bar x\notag\end{equation} \begin{equation}{\Psi ^t}\left( {\varepsilon ,\left( {x,t,u} \right)} \right) = {\text{exp}}\left( {\varepsilon \pmb{v_5}} \right)\left( t \right) = \left( {1 + \varepsilon \pmb{v_5} + \frac{1}{2}{{\left( {\varepsilon \pmb{v_5}} \right)}^2} +  \ldots } \right)\left( t \right) = t + \varepsilon \pmb{v_5}t + \frac{1}{2}{\varepsilon ^2}\pmb{v_5}\left( {\pmb{v_5}t} \right) +  \ldots \notag\end{equation} \begin{equation}{\Psi ^t}\left( {\varepsilon ,\left( {x,t,u} \right)} \right) = t \equiv \bar t\notag\end{equation} \begin{equation}{\Psi ^u}\left( {\varepsilon ,\left( {x,t,u} \right)} \right) = {\text{exp}}\left( {\varepsilon \pmb{v_5}} \right)\left( u \right) = \left( {1 + \varepsilon \pmb{v_5} + \frac{1}{2}{{\left( {\varepsilon \pmb{v_5}} \right)}^2} +  \ldots } \right)\left( u \right) = u + \varepsilon \pmb{v_5}u + \frac{1}{2}{\varepsilon ^2}\pmb{v_5}\left( {\pmb{v_5}u} \right) +  \ldots \notag\end{equation} \begin{equation}{\Psi ^u}\left( {\varepsilon ,\left( {x,t,u} \right)} \right) = u - x\varepsilon u + \frac{1}{2}{\varepsilon ^2}\left( {2t{\partial _x} - xu{\partial _u}} \right)\left( { - xu} \right) +  \ldots  = u\left[ { - x\varepsilon  + \frac{1}{2}{\varepsilon ^2}\left( { - 2t + {x^2}} \right) +  \ldots } \right]\notag\end{equation} Now summing this last component is challenging, so let's resort to the "shortcut" (Olver (1.11)): \begin{equation}\frac{{d{\Psi ^u}}}{{d\varepsilon }} \equiv \frac{{du\left( \varepsilon  \right)}}{{d\varepsilon }} =  - x\left( \varepsilon  \right)u\left( \varepsilon  \right) =  - \left( {{x_0} + 2\varepsilon t} \right)u\left( \varepsilon  \right)\notag\end{equation} \begin{equation}\frac{{du\left( \varepsilon  \right)}}{{u\left( \varepsilon  \right)}} =  - \left( {{x_0} + 2\varepsilon t} \right)d\varepsilon  \to \ln \left( {u\left( \varepsilon  \right)} \right) - \ln \left( {{u_0}} \right) =  - \left( {{x_0}\varepsilon  + {\varepsilon ^2}t} \right)\notag\end{equation} Solving: \begin{equation}u/{u_0} = \exp \left( { - \left( {{x_0}\varepsilon  + {\varepsilon ^2}t} \right)} \right)\notag\end{equation} Replacing $u \to \bar u$ and ${u_0} \to u$: \begin{equation}\bar u = u\exp \left( { - \left( {\varepsilon x + {\varepsilon ^2}t} \right)} \right)\notag\end{equation} So \begin{equation}{G_5}:\;\;\;\left( {x,t,u} \right)\mapsto \left( {\bar x = x + 2\varepsilon t,\bar t = t,\bar u = u \cdot {\text{exp}}\left( { - \varepsilon x - {\varepsilon ^2}t} \right)} \right)\notag\end{equation} Summarizing the process, we applied the prolonged vector to the invariance condition on our PDE to find determining equations for the basic vector coefficients.  Solving them with the condition of the algebraic relations implied by the PDE, we arrive at a number of independent “infinitesimal” vectors that span the symmetry group of the original PDE.  We apply the exponential flow process to determine the full symmetry transforms for each vector.

Armed with the group symmetries for a PDE we can seek sub-manifold solutions, or G-Invariant solutions, the topic of the next section.
