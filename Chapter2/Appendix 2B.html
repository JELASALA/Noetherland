Appendix 2B

Selected 2D Wave Equation Symmetry Vector Commutation Relations

In this appendix we compute selected commutation relations for the symmetry vector generators for the 2D Wave Equation and draw some conclusions about their applicability in finding G-invariant solutions by combining two of them.

Translation and Dilation:

\begin{equation}{\pmb{v}_{xyt}} = a{\partial _x} + b{\partial _y} + c{\partial _t}\notag\end{equation} and \begin{equation}\pmb{d} = x{\partial _x} + y{\partial _y} + t{\partial _t}\notag\end{equation} Employing a dummy function upon which the vectors operate: \begin{equation}\left[ {\pmb{d},{\pmb{v}_{xyt}}} \right]f = \left( {x{\partial _x} + y{\partial _y} + t{\partial _t}} \right)\left( {a{\partial _x}f + b{\partial _y}f + c{\partial _t}f} \right) - \left( {a{\partial _x} + b{\partial _y} + c{\partial _t}} \right)\left( {x{\partial _x}f + y{\partial _y}f + t{\partial _t}f} \right)\hspace{100em}\notag\end{equation} \begin{multline*} = \left( {x{\partial _x}} \right)\left( {a{\partial _x}f + b{\partial _y}f + c{\partial _t}f} \right) + \left( {y{\partial _y}} \right)\left( {a{\partial _x}f + b{\partial _y}f + c{\partial _t}f} \right) + \left( {t{\partial _t}} \right)\left( {a{\partial _x}f + b{\partial _y}f + c{\partial _t}f} \right) \\ - \left( {a{\partial _x}} \right)\left( {x{\partial _x}f + y{\partial _y}f + t{\partial _t}f} \right) - \left( {b{\partial _y}} \right)\left( {x{\partial _x}f + y{\partial _y}f + t{\partial _t}f} \right) - \left( {c{\partial _t}} \right)\left( {x{\partial _x}f + y{\partial _y}f + t{\partial _t}f} \right)\notag\end{multline*} \begin{multline*} = \left( {ax{\partial _{xx}}f + bx{\partial _{xy}}f + cx{\partial _{xt}}f} \right) + \left( {ay{\partial _{xy}}f + by{\partial _{yy}}f + cy{\partial _{ty}}f} \right) + \left( {at{\partial _{xt}}f + bt{\partial _{yt}}f + ct{\partial _{tt}}f} \right) \\ - \left( {a{\partial _x}} \right)\left( {{\partial _x}f + ax{\partial _{xx}}f + ay{\partial _{xy}}f + at{\partial _{xt}}f} \right) - \left( {bx{\partial _{xy}}f + b{\partial _y}f + by{\partial _{yy}}f + bt{\partial _{ty}}f} \right) \\ - \left( {c{\partial _t}} \right)\left( {cx{\partial _{xt}}f + cy{\partial _{yt}}f + c{\partial _t}f + ct{\partial _{tt}}f} \right)\notag\end{multline*} \begin{multline*} = \left( {ax{\partial _{xx}}f + bx{\partial _{xy}}f + cx{\partial _{xt}}f} \right) + \left( {ay{\partial _{xy}}f + by{\partial _{yy}}f + cy{\partial _{ty}}f} \right) + \left( {at{\partial _{xt}}f + bt{\partial _{yt}}f + ct{\partial _{tt}}f} \right)\\ - \left( {a{\partial _x}f + ax{\partial _{xx}}f + ay{\partial _{xy}}f + at{\partial _{xt}}f} \right) - \left( {bx{\partial _{xy}}f + b{\partial _y}f + by{\partial _{yy}}f + bt{\partial _{ty}}f} \right) \\ - \left( {cx{\partial _{xt}}f + cy{\partial _{yt}}f + c{\partial _t}f + ct{\partial _{tt}}f} \right)\notag\end{multline*} \begin{multline*} = \left( {bx{\partial _{xy}}f + cx{\partial _{xt}}f} \right) + \left( {ay{\partial _{xy}}f + cy{\partial _{ty}}f} \right) + \left( {at{\partial _{xt}}f + bt{\partial _{yt}}f} \right) - \left( {a{\partial _x}f + ay{\partial _{xy}}f + at{\partial _{xt}}f} \right) \\ - \left( {bx{\partial _{xy}}f + b{\partial _y}f + bt{\partial _{ty}}f} \right) - \left( {cx{\partial _{xt}}f + cy{\partial _{yt}}f + c{\partial _t}f} \right)\notag\end{multline*} Then \begin{equation}\left[ {\pmb{d},{\pmb{v}_{xyt}}} \right] = - a{\partial _x} - b{\partial _y} - c{\partial _t} = - {v_{xyt}}\notag\end{equation} A subset of this case is \begin{equation}{\pmb{v}_{xy}} = a{\partial _x} + b{\partial _y}\notag\end{equation} with \begin{equation}\pmb{d} = x{\partial _x} + y{\partial _y} + t{\partial _t}\notag\end{equation} whose commutator is obviously: \begin{equation}\left[ {\pmb{d},{\pmb{v}_{xy}}} \right] = - a{\partial _x} - b{\partial _y} - c{\partial _t} = - {v_{xy}}\notag\end{equation} Rotation and Dilation \begin{equation}{\pmb{r}_{xy}} = - y{\partial _x} + x{\partial _y}\notag\end{equation} \begin{equation}\pmb{d} = x{\partial _x} + y{\partial _y} + t{\partial _t}\notag\end{equation} \begin{equation}\left[ {{\pmb{r}_{xy}},\pmb{d}} \right]f = \left( { - y{\partial _x} + x{\partial _y}} \right)\left( {x{\partial _x}f + y{\partial _y}f + t{\partial _t}f} \right) - \left( {x{\partial _x} + y{\partial _y} + t{\partial _t}} \right)\left( { - y{\partial _x}f + x{\partial _y}f} \right)\hspace{100em}\notag\end{equation} \begin{multline*} = \left( { - y{\partial _x}} \right)\left( {x{\partial _x}f + y{\partial _y}f + t{\partial _t}f} \right) + \left( {x{\partial _y}} \right)\left( {x{\partial _x}f + y{\partial _y}f + t{\partial _t}f} \right) - \left( {x{\partial _x} + y{\partial _y} + t{\partial _t}} \right)\left( { - y{\partial _x}f} \right) - \left( {x{\partial _x} + y{\partial _y} + t{\partial _t}} \right)\left( {x{\partial _y}f} \right)\notag\end{multline*} \begin{multline*} = - \left( {y{\partial _x}f + yx{\partial _{xx}}f + yy{\partial _{yx}}f + yt{\partial _{tx}}f} \right) + \left( {xx{\partial _{xy}}f + x{\partial _y}f + xy{\partial _{yy}}f + xt{\partial _{ty}}f} \right) \\ - \left( { - xy{\partial _{xx}}f - y{\partial _x}f - yy{\partial _{xy}}f - yt{\partial _{xt}}f} \right) - \left( {x{\partial _y}f + xx{\partial _{yx}}f + xy{\partial _{yy}}f + xt{\partial _{yt}}f} \right)\notag\end{multline*} \begin{equation}\left[ {{\pmb{r}_{xy}},\pmb{d}} \right] = - \left( {y{\partial _x}} \right) + \left( {x{\partial _y}} \right) - \left( { - y{\partial _x}} \right) - \left( {x{\partial _y}} \right)\notag\end{equation} So \begin{equation}\left[ {{\pmb{r}_{xy}},\pmb{d}} \right] = 0\notag\end{equation} Rotation and Translation \begin{equation}{\pmb{r}_{xy}} = - y{\partial _x} + x{\partial _y}\notag\end{equation} \begin{equation}\pmb{v} = a{\partial _x} + b{\partial _y} + c{\partial _t}\;\notag\end{equation} \begin{equation}\left[ {{\pmb{r}_{xy}},\pmb{v}} \right]f = \left( { - y{\partial _x} + x{\partial _y}} \right)\left( {a{\partial _x}f + b{\partial _y}f + c{\partial _t}f} \right) - \left( {a{\partial _x} + b{\partial _y} + c{\partial _t}} \right)\left( { - y{\partial _x}f + x{\partial _y}f} \right)\hspace{100em}\notag\end{equation} \begin{multline*} = - \left( {ay{\partial _{xx}}f + by{\partial _{yx}}f + cy{\partial _{tx}}f} \right) + \left( {ax{\partial _{xy}}f + bx{\partial _{yy}}f + cx{\partial _{ty}}f} \right) \\ - \left( { - ay{\partial _{xx}}f - b{\partial _x}f - by{\partial _{xy}}f - cy{\partial _{xt}}f} \right) - \left( {a{\partial _y}f + ax{\partial _{yx}}f + xb{\partial _{yy}}f + cx{\partial _{yt}}f} \right)\notag\end{multline*} So \begin{equation}\left[ {{\pmb{r}_{xy}},\pmb{v}} \right] = b{\partial _x} - a{\partial _y}\notag\end{equation} These do not commute nor do they yield a common vector. Even a single dimension translation, say \begin{equation}{\pmb{v}_x} = a{\partial _x}\notag\end{equation} yields \begin{equation}\left[ {{\pmb{r}_{xy}},{\pmb{v}_x}} \right] = - a{\partial _y} \propto {\pmb{v}_y}\notag\end{equation} \begin{equation}{\pmb{v}_y} = b{\partial _y}\notag\end{equation} This is why attempts (not exhibited in the main body of Section 2.7) to employ Rotation (${\pmb{r}_{xy}}$) with Translation (${\pmb{v}_x},{\pmb{v}_y},{\pmb{v}_t})$ yielded null results.

Commutator Table: We collect results as follows:

$\pmb{d}$ ${\pmb{r}_{xy}}$ ${\pmb{v}_{xy}}$
$\pmb{d}$ 0 0 $ - {\pmb{v}_{xy}}$
${\pmb{r}_{xy}}$ 0 0 $b{\pmb{v}_x} - a{\pmb{v}_y}$
${\pmb{v}_{xy}}$ ${\pmb{v}_{xy}}$ $ - b{\pmb{v}_x} + a{\pmb{v}_y}$ 0

We see the dilation vector behaves well with the other two, but the rotation vector does not behave “normally” with the translation vector, producing a result that cannot be constructed from either of the vectors involved.
