As an intermediate step, consider \begin{equation}{\zeta _{xx}} = {\eta _{yx}} = - {\zeta _{yy}}\notag\end{equation} \begin{equation}{\zeta _{tt}} = {c^2}{\tau _{xt}} = {c^2}{\zeta _{xx}}\notag\end{equation} So \begin{equation}{c^2}{\zeta _{xx}} = - {c^2}{\zeta _{yy}} = {\zeta_{tt}}\tag{2A.8a}\label{8a}\end{equation} Similarly \begin{equation}{c^2}{\eta _{xx}} = - {c^2}{\eta _{yy}} = -{\eta_{tt}}\tag{2A.8b}\label{8b}\end{equation} \begin{equation}{c^2}{\tau _{xx}} = {c^2}{\tau _{yy}} = {\tau_{tt}}\tag{2A.8c}\label{8c}\end{equation} Taking third derivatives and playing games like that above, we can show all three coefficients depend at most on the squares (or bilinear combinations) of the independent variables. \begin{equation} \begin{split} \zeta &= {c_1} + {c_4}x - {c_5}y + {c_6}t + {c_8}\left( {{x^2} - {y^2} + {c^2}{t^2}} \right) + 2{c_9}xy + 2{c_{10}}xt\\ \eta &= {c_2} + {c_5}x + {c_4}y + {c_7}t + {c_9}\left( { - {x^2} + {y^2} + {c^2}{t^2}} \right) + 2{c_8}xy + 2{c_{10}}yt\\ \tau &= {c_3} + \frac{{{c_6}x}}{{{c^2}}} + \frac{{{c_7}y}}{{{c^2}}} + {c_4}t + \frac{{{c_{10}}}}{{{c^2}}}\left( {{x^2} + {y^2} + {c^2}{t^2}} \right) + 2{c_9}yt + 2{c_8}xt \end{split} \tag{2A.9}\label{9}\end{equation} Let us address the dependent variable coefficient. Having established earlier$\;\phi $ depends at most linearly on $u$, we employ the form $\phi = \beta \left( {x,y,t} \right)u + \alpha \left( {x,y,t} \right)$, noting after taking the $u$-derivative we have only $\beta$ remaining. Substituted into \eqref{7} we obtain Olver’s equations on page 124 with proper dimensions: \begin{equation}{c^2}2{\beta _x} = {c^2}\left( {{\zeta _{xx}} + {\zeta _{yy}}} \right) - {\zeta _{tt}}\notag\end{equation} \begin{equation}{c^2}2{\beta _y} = {c^2}\left( {{\eta _{xx}} + {\eta _{yy}}} \right) - {\eta _{tt}}\notag\end{equation} \begin{equation}2{\beta _t} = - {c^2}\left( {{\tau _{xx}} + {\tau _{yy}}} \right) + {\tau _{tt}}\notag\end{equation} Using \eqref{8a} in the first we have: \begin{equation}{c^2}2{\beta _x} = {\zeta _{tt}} + {c^2}{\zeta _{yy}} - {\zeta _{tt}} = {c^2}{\zeta _{yy}}\notag\end{equation} so \begin{equation}2{\beta _x} = {\zeta _{yy}} \notag\end{equation} Similarly using \eqref{8b} and \eqref{8b} in the others: \begin{equation}2{\beta _y} = {\eta _{xx}} = - {\eta _{yy}}\notag\end{equation} \begin{equation}2{\beta _t} = - {c^2}{\tau _{xx}} = - {\tau _{tt}}\notag\end{equation} Employing our functional forms found in \eqref{9} above: \begin{equation}2{\beta _x} = {\zeta _{yy}} = - 2{c_8} \to \notag\end{equation} \begin{equation}\beta = - {c_8}x + f\left( {t,y} \right)\notag\end{equation} Similarly \begin{equation}\beta = - {c_9}y + g\left( {t,x} \right)\notag\end{equation} \begin{equation}\beta = - {c_{10}}t + y\left( {x,y} \right)\notag\end{equation} Combining: \begin{equation}\beta = {c_{11}} - {c_8}x - {c_9}y - {c_{10}}t\notag\end{equation} For $\alpha $ we insert our functional form for $\phi$ \begin{equation}\phi = \beta \left( {x,y,t} \right)u + \alpha \left( {x,y,t} \right)\notag\end{equation} into \eqref{6}: \begin{equation}{\phi _{tt}} - {c^2}\left( {{\phi _{xx}} + {\phi _{yy}}} \right) = 0\notag\end{equation} leaving \begin{equation}{\alpha _{tt}} - {c^2}\left( {{\alpha _{xx}} + \phi {\alpha _{yy}}} \right) = 0\notag\end{equation} showing $\alpha$ satisfies the PDE. Summarizing: \begin{equation} \begin{split} \zeta &= {c_1} + {c_4}x - {c_5}y + {c_6}t + {c_8}\left( {{x^2} - {y^2} + {c^2}{t^2}} \right) + 2{c_9}xy + 2{c_{10}}xt\\ \eta &= {c_2} + {c_5}x + {c_4}y + {c_7}t + {c_9}\left( { - {x^2} + {y^2} + {c^2}{t^2}} \right) + 2{c_8}xy + 2{c_{10}}yt\\ \tau &= {c_3} + \frac{{{c_6}x}}{{{c^2}}} + \frac{{{c_7}y}}{{{c^2}}} + {c_4}t + \frac{{{c_{10}}}}{{{c^2}}}\left( {{x^2} + {y^2} + {c^2}{t^2}} \right) + 2{c_9}yt + 2{c_8}xt \end{split} \notag\end{equation} \begin{equation}\phi = \beta \left( {x,y,t} \right)u + \alpha \left( {x,y,t} \right)\notag\end{equation} with \begin{equation}\beta = {c_{11}} - {c_8}x - {c_9}y - {c_{10}}t\notag\end{equation} and \begin{equation}\alpha = {\alpha _{2DWE}}\notag\end{equation} is any solution to the 2D Wave Equation.
