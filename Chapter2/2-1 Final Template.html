2.1. Motivation

We begin with the vector approach to finding symmetries of physical systems describable by PDEs.  [cite Olver 1986].  The idea is that requiring the PDE function, symbolized as $\Delta $, to vanish defines a sub-manifold (aka sub-variety) Olver calls ${\mathcal{S}_\Delta }$ in the appropriate jet space.  A symmetry of the system of differential equations $\Delta $ is a transformation which maps solutions to solutions; that is, application of the transformation keeps you in the submanifold of solutions (think of a partial rotation around the surface of a sphere). The most basic type of symmetry is a group of point transformations on the space of independent and dependent variables.  We typically restrict attention to fiber-preserving transformations, in which action on independent variables does not depend on dependent variables (aka projectable transformations).  [Olver p.93] [cf General transformations involving derivatives]

Teaser Example:  Let us consider the 1-D (dimensionless) diffusion (heat) equation as a simple example to introduce the concept of symmetry transform. \begin{equation}\Delta \left( {x,t,{u^{\left( 2 \right)}}} \right) = {u_t} - {u_{xx}} = 0\notag\end{equation} Here ${u^{\left( 2 \right)}}$ refers to the highest order partial derivatives of u in our jet space.  The vast machinery to be employed later will give us the set of transformations that "span" the symmetries of the heat equation.  Here is the simplest: \begin{equation}{G_1}:\;\;\;\left( {x,t,u} \right)\centerdot\left( {\bar x = x + \varepsilon ,\bar t = t,\bar u = u} \right)\notag\end{equation} which is associated with the symmetry vector: \begin{equation}\boldsymbol{\v} = \frac{\partial }{{\partial x}}\notag\end{equation} Though this is a trivial example, we want to demonstrate the full meaning of such a transformation and the means to test it.  We need to compute the action of this transform on a "known" solution $u = f\left( {x,t} \right)$ to the PDE.  If $f\left( {x,t} \right)$ is such a solution then \begin{equation}g\centerdot f \equiv \bar f\notag\end{equation} is also supposed to be a solution, where the operation $g$ accomplishes \begin{equation}g\centerdot \left( {x,t,\;u} \right) = \left( {\bar x,\bar t,\bar u} \right)\notag\end{equation} where $\bar u = \bar f\left( {\bar x,\bar t} \right)$.  This is a set of equations: \begin{equation}\bar x = x + \varepsilon \notag\end{equation} \begin{equation}\bar t = t\notag\end{equation} \begin{equation}\bar u = \bar u\left( {\bar x,\bar t} \right)\notag\end{equation} We need to eliminate the original independent variables in favor of the transformed variables, simply accomplished in this case: \begin{equation}\bar u = \bar u\left( {\bar x,\bar t} \right) = u\left( {\bar x - \varepsilon ,\bar t} \right)\notag\end{equation} Does this transformed "solution" still satisfy the original PDE (heat equation)?  That is, if \begin{equation}{u_t} - {u_{xx}} = 0\notag\end{equation} does \begin{equation}{\bar u_{\bar t}} - {\bar u_{\bar x\bar x}} = ? = 0\notag\end{equation} We define auxiliary variables $r \equiv \bar x - \varepsilon $ and $s = \bar t$ which makes application of the chain rule a little more transparent \begin{equation}{\bar u_{\bar x}} = \frac{{\partial \bar u}}{{\partial \bar x}} = \frac{{\partial u\left( {r,s} \right)}}{{\partial \bar x}} = \frac{{\partial u}}{{\partial r}}\frac{{\partial r}}{{\partial \bar x}} + \frac{{\partial u}}{{\partial s}}\frac{{\partial s}}{{\partial \bar x}}\;\notag\end{equation} The last term is zero as $s = \bar t$ is independent of $\bar x$ as they are both independent variables in the transformed chart. \begin{equation}{\bar u_{\bar x}} = {u_r}\left( 1 \right)+0\notag\end{equation} Similarly: \begin{equation}{\bar u_{\bar x\bar x}} = {u_{rr}}\notag\end{equation} and trivially: \begin{equation}{\bar u_{\bar t}} = \frac{{\partial u\left( {r,s} \right)}}{{\partial \bar t}} = \frac{{\partial u}}{{\partial r}}\frac{{\partial r}}{{\partial \bar t}} + \frac{{\partial u}}{{\partial s}}\frac{{\partial s}}{{\partial \bar t}} = 0 + {u_s}\left( 1 \right)\notag\end{equation} Referring back to our variable transform relations $r = \bar x - \varepsilon  = x$ and $s = \bar t = t$ so \begin{equation}{\bar u_{\bar x\bar x}} = {u_{rr}} = {u_{xx}}\;\notag\end{equation} and \begin{equation}{\bar u_{\bar t}} = {u_s} = {u_t}\notag\end{equation} so the transformed solution also obeys the heat equation if the original solution did.  In more complicated cases, where for example $u$ itself is transformed, the "inversion" is more challenging.  A general formula (really a procedure) is given by: \begin{equation}g\circ f = \left[ {{\Phi _g} \circ \left( {\mathbb{1} \times f} \right)} \right] \circ {\left[ {{\Xi _g} \circ \left( {\mathbb{1} \times f} \right)} \right]^{ - 1}}\tag{Olver(2.14)}\end{equation} where the new symbols are defined by: \begin{equation}g\circ \left( {x,u} \right) = \left( {\bar x,\bar u} \right) \equiv \left( {{\Xi _g}\left( {x,u} \right),{\Phi _g}\left( {x,u} \right)} \right)\notag\end{equation} so: \begin{equation}\bar x \equiv {\Xi _g}\left( {x,f\left( x \right)} \right) = {\Xi _g} \circ \left( {\mathbb{1} \times f} \right)\left( x \right)\notag\end{equation} \begin{equation}\bar u \equiv {\Phi _g}\left( {x,f\left( x \right)} \right) = {\Phi _g} \circ \left( {\mathbb{1} \times f} \right)\left( x \right)\notag\end{equation} and the inverse symbol on the last square bracket simply means invert the transformed coordinates before composing with the transformed dependent variable(s).  Note we are employing Olver’s condensed notation in which, for example, ${\Xi _g}$ will represent all the transformed independent variables $\bar x$, appropriately indexed.  Another example will illuminate the process.

Advanced Teaser  Let's try another more complex symmetry transform of the 1D heat equation (obtained later in Section 2.5): \begin{equation}{G_5}:\;\;\;\left( {x,t,u} \right)\circ \left( {\bar x = x + 2\varepsilon t,\bar t = t,\bar u = u\circ exp\left( { - \varepsilon x - {\varepsilon ^2}t} \right)} \right)\notag\end{equation} Since this involves the dependent variable u we need to be more careful, and refer to Olver’s equation (2.14), to which he points when interpreting group solutions.  So \begin{equation}{\Xi ^1}_g\left( {x,t,f\left( {x,t} \right)} \right) = x + 2\varepsilon t \equiv \bar x\notag\end{equation} \begin{equation}{\Xi ^2}_g\left( {x,t,f\left( {x,t} \right)} \right) = t \equiv \bar t\notag\end{equation} \begin{equation}{\Phi _g}\left( {x,t,f\left( {x,t} \right)} \right) = f\left( {x,t} \right)\circ {\text{exp}}\left( { - \varepsilon x - {\varepsilon ^2}t} \right) \equiv \bar u\notag\end{equation} Our job is now to compute the transformed "solution" using \begin{equation}g\circ f = {\Phi _g}\left( {x,t,f\left( {x,t} \right)} \right) \circ {\left[ {{\Xi ^1}_g\left( {x,t,f\left( {x,t} \right)} \right),{\Xi ^2}_g\left( {x,t,f\left( {x,t} \right)} \right)} \right]^{ - 1}}\tag{Olver(2.14)}\notag\end{equation} Note the inverse superscript simply means to invert each of the elements.   \begin{equation}t = \bar t\notag\end{equation} \begin{equation}x = \bar x - 2\varepsilon t = \bar x - 2\varepsilon \bar t\notag\end{equation} So \begin{equation}g\circ f = {\Phi _g} \circ \left[ {\bar x - 2\varepsilon \bar t,\bar t} \right] = f\left( {x,t} \right)\circ {\text{exp}}\left( { - \varepsilon x - {\varepsilon ^2}t} \right) \circ \left[ {\bar x - 2\varepsilon \bar t,\bar t} \right]\notag\end{equation} Composition means everywhere there is an $x\;{\text{or}}\;t$ in the first function replace it with the respective positional function in square brackets on the right of the composition symbol. \begin{equation}g\circ f = f\left( {\bar x - 2\varepsilon \bar t,\bar t} \right)\circ {\text{exp}}\left( { - \varepsilon \left( {\bar x - 2\varepsilon \bar t} \right) - {\varepsilon ^2}\bar t} \right) = f\left( {\bar x - 2\varepsilon \bar t,\bar t} \right)\circ {\text{exp}}\left( { - \varepsilon \bar x + {\varepsilon ^2}\bar t} \right) \equiv \bar f\notag\end{equation} Compare this to the previous example: \begin{equation}\bar u = \bar u\left( {\bar x,\bar t} \right) = u\left( {\bar x - \varepsilon ,\bar t} \right)\notag\end{equation} Aside from the symbolic substitution of $f$ with $u$ these have the same form, with the original "solution" function taking modified "$x$" and "$t$" variables.  Let's replace $f$ with $u$ in our current example and also make a useful symbolic replacement for the arguments in $f$: \begin{equation}\bar u = \bar u\left( {\bar x,\bar t} \right) = u\left( {\bar x - 2\varepsilon \bar t,\bar t} \right)\circ {\text{exp}}\left( { - \varepsilon \bar x + {\varepsilon ^2}\bar t} \right) \equiv u\left( {r\left( {\bar x,\bar t} \right),s\left( {\bar x,\bar t} \right)} \right)\circ {\text{exp}}\left( { - \varepsilon \bar x + {\varepsilon ^2}\bar t} \right)\notag\end{equation} where we again define auxiliary variables $r \equiv \bar x - 2\varepsilon \bar t\;{\text{and}}\;s \equiv \bar t$ to avoid confustion.  Computing: \begin{equation}{\bar u_{\bar x}} = \left[ {\frac{{\partial u\left( {r,s} \right)}}{{\partial r}}\frac{{\partial r}}{{\partial \bar x}} + \frac{{\partial u\left( {r,s} \right)}}{{\partial s}}\frac{{\partial s}}{{\partial \bar x}}} \right]{e^{ - \varepsilon \bar x + {\varepsilon ^2}\bar t}} - \varepsilon u\left( {r,s} \right){e^{ - \varepsilon \bar x + {\varepsilon ^2}\bar t}}\notag\end{equation} \begin{equation}{\bar u_{\bar x}} = {u_r}{e^{ - \varepsilon \bar x + {\varepsilon ^2}\bar t}} - \varepsilon u\left( {r,s} \right){e^{ - \varepsilon \bar x + {\varepsilon ^2}\bar t}}\notag\end{equation} Similarly, \begin{equation}{\bar u_{\bar x\bar x}} = {u_{rr}}{e^{ - \varepsilon \bar x + {\varepsilon ^2}\bar t}} - 2\varepsilon {u_r}{e^{ - \varepsilon \bar x + {\varepsilon ^2}\bar t}} + {\varepsilon ^2}u{e^{ - \varepsilon \bar x + {\varepsilon ^2}\bar t}}\notag\end{equation} and \begin{equation}{\bar u_{\bar t}} = \left[ {{u_r}\frac{{\partial r}}{{\partial \bar t}} + {u_s}\frac{{\partial s}}{{\partial \bar t}}} \right]{e^{ - \varepsilon \bar x + {\varepsilon ^2}\bar t}} + {\varepsilon ^2}u{e^{ - \varepsilon \bar x + {\varepsilon ^2}\bar t}}\notag\end{equation} \begin{equation}{\bar u_{\bar t}} = \left( { - 2\varepsilon {u_r} + {u_s}} \right){e^{ - \varepsilon \bar x + {\varepsilon ^2}\bar t}} + {\varepsilon ^2}u{e^{ - \varepsilon \bar x + {\varepsilon ^2}\bar t}}\notag\end{equation} Combining \begin{equation}{\bar u_{\bar t}} - {\bar u_{\bar x\bar x}} = \left( { - 2\varepsilon {u_r} + {u_s}} \right){e^{ - \varepsilon \bar x + {\varepsilon ^2}\bar t}} + {\varepsilon ^2}u{e^{ - \varepsilon \bar x + {\varepsilon ^2}\bar t}} - {u_{rr}}{e^{ - \varepsilon \bar x + {\varepsilon ^2}\bar t}} + 2\varepsilon {u_r}{e^{ - \varepsilon \bar x + {\varepsilon ^2}\bar t}} - {\varepsilon ^2}u{e^{ - \varepsilon \bar x + {\varepsilon ^2}\bar t}}\notag\end{equation} \begin{equation}{\bar u_{\bar t}} - {\bar u_{\bar x\bar x}} = \left( {{u_s} - {u_{rr}}} \right){e^{ - \varepsilon \bar x + {\varepsilon ^2}\bar t}}\notag\end{equation} Now as in the first example, we recognize \begin{equation}r = \bar x - 2\varepsilon \bar t = x\;\;{\text{and}}\;s\; = \bar t = t\notag\end{equation} so \begin{equation}{\bar u_{\bar t}} - {\bar u_{\bar x\bar x}} = \left( {{u_t} - {u_{xx}}} \right){e^{ - \varepsilon \bar x + {\varepsilon ^2}\bar t}} = 0\notag\end{equation} verifying the transformed solution satisfies the heat equation.  Notice now that we generate actual solutions to the heat equation once we have even a trivial solution to start with.  For example, the substitution \begin{equation}u = A = constant\notag\end{equation} certainly satisfies the heat equation, so \begin{equation}\bar u = A{e^{ - \varepsilon \bar x + {\varepsilon ^2}\bar t}}\notag\end{equation} is also a solution of the heat equation.  Hopefully this "teaser" has stimulated your interest in how to find these symmetry groups from the original PDE system.  In effect, without explicitly solving the PDE, we can find "orbitals" within the solution surface that satisfy the PDE.  To accomplish this from scratch we go back a bit in Olver to the concept of infinitesimal vector generators and apply them to algebraic equations first.
