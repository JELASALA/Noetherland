3.5 Similarity Solutions

Having shown how to find isovectors for a given PDE (system) using EDS techniques we now utilize this information to find so-called similarity solutions, equivalent to Olver’s G-invariant solutions.  Suhubi’s process is based on his equation (9.5.3) – symbolically modified to conform to our conventions:

\begin{equation}{\phi ^*}\left\langle {V,{\sigma ^\alpha }} \right\rangle  = 0\label{A}\end{equation}

where ${\sigma ^\alpha }$ is the “zeroeth order” contact form for the PDE (in this section we assume $N = 1$ so drop the Greek index in what follows).  Here $\phi $ is a mapping from the Graph space to the contact manifold in Kinematic space so ${\phi ^*}$ pulls back to the Graph space. The origin of this condition is explained in the dropdown below.

We suppose $\phi \left( x \right) = u$ corresponds to a solution to a PDE under consideration that remains invariant along the flow produced by an isovector $V$ of that PDE.  Form the function $f = \phi \left( x \right) - u = 0$ and demand its Lie derivative with respect to $V$ be invariant.  That is, the difference between $\phi $ and $u$ that starts out at zero remains zero while the functions are transported by $V$ in Kinematic space.  Thus ${\mathcal{L}_V}\left( f \right) = V\left( f \right) = 0$.  Let the vector be represented as: \begin{equation}V = {X^i}\left( {x,\phi \left( x \right),{\phi _j}\left( x \right)} \right){\partial _{{x^i}}} + U\left( {x,\phi \left( x \right),{\phi _j}\left( x \right)} \right){\partial _u}\notag\end{equation}Then the condition is \begin{equation}V\left( f \right) = \left( {{X^i}{\partial _{{x^i}}} + U{\partial _u}} \right)\left( {\phi \left( x \right) - u} \right) = {X^i}\frac{{\partial \phi \left( x \right)}}{{\partial {x^i}}} - U = 0\notag\end{equation} Recall the zeroeth order contact form has the structure (for $N = 1$) in Kinematic space: \begin{equation}\sigma  = du - {v_i}d{x^i}\notag\end{equation} Thus the condition above can be represented as \begin{equation}\left\langle {V,\sigma } \right\rangle  = \left\langle {{X^i}{\partial _{{x^i}}} + U{\partial _u},du - {v_i}d{x^i}} \right\rangle  = U - {X^i}{v_i}\notag\end{equation}so \begin{equation}{\phi ^*}\left\langle {V,\sigma } \right\rangle  = U - {X^i}{u_i} = 0\notag\end{equation}

Let us apply this to our canonical example the 1D heat equation:

\begin{equation}{u_{xx}} - {u_t} = 0\notag\end{equation}we have \begin{equation}\sigma  = du - {v_1}dx - {v_2}dt\notag\end{equation} where \begin{equation}{\phi ^*}{v_1} = {u_x}\notag\end{equation}\begin{equation}{\phi ^*}{v_2} = {u_t}\notag\end{equation} Let us find the similarity solution under the Galilean Boost vector used at the end of Section 2.6: \begin{equation}V = 2t{\partial _x} - xu{\partial _u}\notag\end{equation}The condition \begin{equation}\left\langle {V,{\sigma ^\alpha }} \right\rangle  = 0\notag\end{equation}leads to \begin{equation}\left\langle {V,\sigma } \right\rangle  = \left\langle {2t{\partial _x} - xu{\partial _u},du - {v_1}dx - {v_2}dt} \right\rangle  =  - 2t{v_1} - xu = 0\notag\end{equation} Applying ${\phi ^*}\;$turns ${v_1}$ into ${\partial _x}u$:
\begin{equation}2t\frac{{\partial u}}{{\partial x}} + xu = 0\notag\end{equation}We may integrate this directly: \begin{equation} - \frac{{xdx}}{{2t}} = \frac{{du}}{u} \to - \frac{{{x^2}}}{{4t}} = \ln \left( u \right) - \ln \left( {{g_1}} \right)\notag\end{equation} where we allow \begin{equation}{g_1} = {g_1}\left( t \right)\;{\text{at most}}\notag\end{equation} treating $t$ as a parameter on the invariant surface. \begin{equation} \to \exp \left( { - \frac{{{x^2}}}{{4t}}} \right) = \frac{u}{{{g_1}}}\notag\end{equation}so our first (and only) invariant is: \begin{equation}{g_1} = u\exp \left( {\frac{{{x^2}}}{{4t}}} \right)\notag\end{equation}and our provisional solution is: \begin{equation}u = {g_1}\exp \left( { - \frac{{{x^2}}}{{4t}}} \right)\notag\end{equation}What must ${g_1}$ look like to satisfy the heat equation?  \begin{equation}{u_t} = {u_{xx}}\notag\end{equation} \begin{equation}{u_t} = \exp \left( { - \frac{{{x^2}}}{{4t}}} \right){\partial _t}{g_1} + {g_1}{\partial _t}\exp \left( { - \frac{{{x^2}}}{{4t}}} \right) = \left[ {{\partial _t}{g_1} + {g_1}\left( {\frac{{{x^2}}}{{4{t^2}}}} \right)} \right]\exp \left( { - \frac{{{x^2}}}{{4t}}} \right)\notag\end{equation} \begin{equation}{u_x} = {g_1}{\partial _x}\exp \left( { - \frac{{{x^2}}}{{4t}}} \right) =  - {g_1}\left( {\frac{x}{{2t}}} \right)\exp \left( { - \frac{{{x^2}}}{{4t}}} \right)\notag\end{equation} \begin{equation}{u_{xx}} = \left[ { - {g_1}\left( {\frac{1}{{2t}}} \right) + {g_1}\left( {\frac{x}{{2t}}} \right)\left( {\frac{x}{{2t}}} \right)} \right]\exp \left( { - \frac{{{x^2}}}{{4t}}} \right)\notag\end{equation} Combining and dropping common factors: \begin{equation}{\partial _t}{g_1} + {g_1}\left( {\frac{{{x^2}}}{{4{t^2}}}} \right) =  - {g_1}\left( {\frac{1}{{2t}}} \right) + {g_1}\left( {\frac{x}{{2t}}} \right)\left( {\frac{x}{{2t}}} \right)\notag\end{equation} \begin{equation}{\partial _t}{g_1} + \frac{1}{{2t}}{g_1} = 0\notag\end{equation} We can integrate this directly.  I will replace the partial derivative by a total derivative given the presumed dependency of ${g_1}$ on $t$ alone: \begin{equation}\frac{{d{g_1}}}{{{g_1}}} =  - \frac{{dt}}{{2t}} \to \ln \left( {{g_1}} \right) =  - \frac{1}{2}\ln \left( t \right) + \ln \left( k \right)\notag\end{equation}so \begin{equation}{g_1}\sqrt t  = k\notag\end{equation}yielding: \begin{equation}u = \frac{k}{{\sqrt t }}\exp \left( { - \frac{{{x^2}}}{{4t}}} \right) \;\color{blue}\blacksquare \notag\end{equation} in agreement with the vector-based G-invariant solution for this isovector found in Section 2.6 but with far less computation.

Note:  in the above example we had only one invariant function, found with a straightforward integration.  For PDEs with $n > 2$ we will find it useful to employ the Method of Characteristics to discover the invariant functions, a highly useful technique elaborated upon in Appendix 1.9A.
