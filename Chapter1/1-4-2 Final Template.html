1.4.2  Composition of Curves and Functions

The diagram below shows our example - an orange parabola for the curve-path and a black circle for the function-field intersecting at $\left( {1,1} \right)$:


You may envision an observer walking along that curve-path and "observing" or "sampling" the function-field he passes through.

We begin by assuming all of the mathematical objects we deal with are analytic – which means they can be expressed as a power series and specifically for our purpose a Taylor series.  So coordinates, curve-paths, and paths through function-fields as polynomials using a Taylor series.  We have already assumed such analyticity in Section 3 when we employed Taylor expansions and power series in conjunction with our analysis of the rotation transformation group.

In Section 1.2 we noted that curve-paths are maps from the real numbers (symbolized by the parameter $s$) to the manifold coordinates, such as

\begin{equation}\gamma |\;\mathbb{R}\cdot M\;{\text{as}}\;\gamma \left( s \right)\cdot \left\{ {x\left( s \right),y\left( s \right)} \right\}\notag\end{equation} The parametric equations for the orange parabola in the figure above are: \begin{equation}x\left( s \right) = s +{x_o}\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;y\left( s \right) = s\left( {s + 2{x_o}} \right) + {y_o}\notag\end{equation}

where we take notice that ${x_o}$ and ${y_o}$ are the values of the coordinates for the parameter value $s = 0$.


For a curve-path to “sample” a field-function we perform a composition of the two.  Symbolically this is represented by:

\begin{equation}\left( {f \circ \gamma } \right)\left( s \right) = f\left( {x\left( s \right),y\left( s \right)} \right)\notag\end{equation} The function-field starts out as a function of the variables $x$ and $y$ and after composing it with a curve it is expressible as a function of the parameter used by the curve.  The result is that the function “reports” its value along the parameter-driven curve, which is what we want.  Technically we should not use the same symbol $f$ for the composition as its functionality has changed thanks to the curve parameterization. For our ongoing example \begin{equation}x\left( s \right) = s + {x_o};\;\;\;\;\;\;y\left( s \right) = s\left( {s + 2{x_o}} \right) + {y_o}\notag\end{equation} \begin{equation}f\left( {x,y} \right) = {x^2} + {y^2}\notag\end{equation} we can directly calculate the composition as follows: \begin{equation}f\left( {x\left( s \right),y\left( s \right)} \right) = {\left( {s + {x_o}} \right)^2} + {\left( {s\left( {s + 2{x_o}} \right) + {y_o}} \right)^2}\notag\end{equation} \begin{equation}\tilde f\left( s \right) = {s^4} + 4{x_o}{s^3} + \left( {1 + 4{x_o}^2 + 2{y_o}} \right){s^2} + \left( {2{x_o} + 4{x_o}{y_o}} \right)s + \left( {{x_o}^2 + {y_o}^2} \right)\notag\end{equation} This macroscopic/global result is very messy yet provides the value of our function-field all along the parabolic path.  Clearly its value is not constant as it would be if we followed one of the invariants (contours) of the function.  Let us instead sample the function incrementally/microscopically/locally along the curve starting at the point $\left( {{x_o},{y_o}} \right)$, which suggests we employ a Taylor series to find first order information from the outset.  Let us first expand the function to first order: \begin{equation}f\left( {x,y} \right) \cong f\left( {{x_o},{y_o}} \right) + {\left( {\frac{{\partial f}}{{\partial x}}} \right)_o}\left( {x - {x_o}} \right) + {\left( {\frac{{\partial f}}{{\partial y}}} \right)_o}\left( {y - {y_o}} \right) +  \ldots \notag\end{equation} We can now express the independent variables in terms of the parameter $s$ to compute the composition and expand them as well using a Taylor series to first order. \begin{equation}\left( {f \circ \gamma } \right)\left( s \right) = f\left( {x\left( s \right),y\left( s \right)} \right) \cong f\left( {x\left( 0 \right),y\left( 0 \right)} \right) + {\left( {\frac{{\partial f}}{{\partial x}}} \right)_{s = o}}\left( {x\left( s \right) - {x_o}} \right) + {\left( {\frac{{\partial f}}{{\partial y}}} \right)_{s = o}}\left( {y\left( s \right) - {y_o}} \right) \notag\end{equation} \begin{equation}\cong f\left( {x\left( 0 \right),y\left( 0 \right)} \right) + {\left( {\frac{{\partial f}}{{\partial x}}} \right)_{s = o}}\left[ {\left( {{x_o} + {{\left( {\frac{{dx}}{{ds}}} \right)}_{s = o}}s + \ldots } \right) - {x_o}} \right] + {\left( {\frac{{\partial f}}{{\partial y}}} \right)_{s = o}}\left[ {\left( {{y_o} + {{\left( {\frac{{dy}}{{ds}}} \right)}_{s = o}}s + \ldots } \right) - {y_o}} \right] \notag\end{equation} \begin{equation}= f\left( {x\left( 0 \right),y\left( 0 \right)} \right) + s{\left( {\frac{{\partial f}}{{\partial x}}} \right)_{s = o}}{\left( {\frac{{dx}}{{ds}}} \right)_{s = o}} + s{\left( {\frac{{\partial f}}{{\partial y}}} \right)_{s = o}}{\left( {\frac{{dy}}{{ds}}} \right)_{s = o}}\notag\end{equation} \begin{equation}f\left( {x\left( s \right),y\left( s \right)} \right) \cong f\left( {{x_o},{y_o}} \right) + s{\left( {\frac{{\partial f}}{{\partial x}}\frac{{dx}}{{ds}} + \frac{{\partial f}}{{\partial y}}\frac{{dy}}{{ds}}} \right)_{s = o}}\notag\end{equation} Let us employ the function and curve used earlier to calculate the LLA for this composition.

\begin{equation}x\left( s \right) = s + {x_o}; \;\;\;\; y\left( s \right) = s\left( {s + 2{x_o}} \right) + {y_o}\notag\end{equation} \begin{equation}f\left( {x,y} \right) = {x^2} + {y^2}\notag\end{equation} Then \begin{equation}f\left( {x\left( s \right),y\left( s \right)} \right) \cong f\left( {{x_o},{y_o}} \right) + s{\left( {2x1 + 2y\left( {2s + 2{x_o}} \right)} \right)_{s = o}}\notag\end{equation} \begin{equation}\left( {f \circ \gamma } \right)\left( s \right) = f\left( {x\left( s \right),y\left( s \right)} \right) \cong f\left( {{x_o},{y_o}} \right) + s\left( {2{x_o} + 4{x_o}{y_o}} \right)\notag\end{equation} This local linear approximation (LLA) to the composition agrees with the full expression above in the limit of infinitesimal $s$.  Now focusing on the infinitesimal interaction may seem severely limiting, yet Lie’s work tells us we can learn almost??? everything we need to know by examining the LLA behavior.

Reflecting on the above image one more time, the orange-colored curve is the curve-path $\gamma \left( s \right)$ that takes a real number $s$ and produces a set of continuous points whose coordinates are $x\left( s \right) = s + {x_o}\;{\text{and}}\;y\left( s \right) = s\left( {s + 2{x_o}} \right) + {y_o}$. The function-field map f$\left( {x,y} \right) = {x^2} + {y^2}$ takes a set of coordinates and produces a real number and one of its invariant-value contours is the black circle.  The curve-path and the function-field contour intersect at the point $\left( {{x_o},{y_o}} \right) = \left( {1,1} \right)$ where $s = 0$.   Our algebraic depiction of the composition of the function-field with the curve-path at the local linear approximation level is:

\begin{equation}\left( {f \circ \gamma } \right)\left( s \right) = f\left( {x\left( s \right),y\left( s \right)} \right) \cong f\left( {{x_o},{y_o}} \right) + s{\left( {\frac{{\partial f}}{{\partial x}}\frac{{dx}}{{ds}} + \frac{{\partial f}}{{\partial y}}\frac{{dy}}{{ds}}} \right)_o}\notag\end{equation} The rightmost term in parentheses is the LLA to the composition and contains the infinitesimal approximation to the interaction between the curve-path and the function-field at the point $\left( {{x_o},{y_o}} \right)$.  Let's examine that term in a bit more detail: \begin{equation}\frac{{\partial f}}{{\partial x}}\frac{{dx}}{{ds}} + \frac{{\partial f}}{{\partial y}}\frac{{dy}}{{ds}} = \frac{{dx}}{{ds}}\frac{{\partial f}}{{\partial x}} + \frac{{dy}}{{ds}}\frac{{\partial f}}{{\partial y}} = \left( {\frac{{dx}}{{ds}},\frac{{dy}}{{ds}}} \right)\cdot \left( {\frac{{\partial f}}{{\partial x}},\frac{{\partial f}}{{\partial y}}} \right) \to \vec V\cdot \vec \nabla f\notag\end{equation} This is the directional derivative in "normal" Euclidean 2-space and the two "vectors" have well-known geometrical representations.   Notionally, since the gradient of a function points in the direction of maximum increase, dotting it with another vector gives the increase of the function in the direction of the vector.  As we move forward in developing the mathematics for this tutorial these “elements” will take on new and more powerful representations.  As constructed, both of these elements are LLAs, one of the curve-path map and the other of the function-field map.  [MAP?] These LLAs are "dotted" together to produce a scalar result in Euclidean 2-space.

The next conceptual leap is integral to a deep understanding of the methodologies to be used in future chapters of our e-tutorial.
