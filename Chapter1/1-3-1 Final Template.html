Section 1.3.1

Continuous Groups

Group theory is one of the cornerstones of modern mathematics and science with an astonishing number of applications.  There is a vast history of its development which the interested student may find in Appendix 1A. Generally there are two kinds:  discrete and continuous.  To realize a group we need a set of elements and some kind of operation between them.  Then we specify properties that must be preserved under that operation.

In the discrete case, we picture a perfect crystal.  Elements are fundamental building blocks called unit cells and operations are rotations, displacements, reflections and inversions.  For example, a diamond possesses cubic symmetry, with three 4-fold axes, four 3-fold axes, six 2-fold axes, nine mirror planes, and an inversion center.  A very nice discussion of crystal symmetries is found at https://opengeology.org/Mineralogy/10-crystal-morphology-and-symmetry/.

In this course we are exclusively focused on continuous groups.  Above we introduced an example of a continuous translation group.  We will use the kind of groups that Sophus Lie specialized in, namely continuous transformation groups or simply Lie Groups.  In particular, to quote Peter Olver (1986, p.75):  we seek ”the largest local group of transformations acting on the independent and dependent variables of the system [being examined] with the property that it transform solutions of the system to other solutions.”  This is the kind of symmetry we seek, inextricably connected to continuous group behaviors. 

All groups must have the following four properties:
            1) closure:  if you operate any element "on" any other you have to obtain another element in the set.  Consider the set of all integers under the operation of addition: $6 + 37 = 43$ and yes 43 is in the set of integers.
            2) associativity:  if you operate with three elements a, b, and c then the result of $\left( {a\cdot b} \right)\cdot c$
must equal $a\cdot \left( {b\cdot c} \right)$.  Note: we use the “dot” to stand for the relevant operation for the group under consideration, that would be the “+” operation for the integers under addition.  Integer addition (and multiplication) obey this property: $\left( {6 + 30} \right) + 7 = 6 + \left( {30 + 7} \right) = 43$.  Note we do NOT demand freedom for order of elements, i.e., commutivity (though integers under addition do have this property).
            3)  identity:  there has to be an element that when used in an operation does nothing. For integer addition we must include zero element in the set.
            4) inverse: every element must have an inverse, i.e. $a\cdot {a^{ - 1}} = 0$, the identity element.  For integers under addition we must include negative numbers to satisfy this requirement: $43 + \left( { - 43} \right) = 0$.
