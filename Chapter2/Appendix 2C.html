Appendix 2C

2D Wave Equation Single Vector Reduced PDEs

When only a single symmetry vector for the 2D Wave Equation is applied we reduce the $p = 3$ PDE to simpler one containing only two independent variables. This appendix is best viewed after going through the main body of Section 2.7 as we draw on some of those results for brevity.

Example A: Translation (spatial) only. \begin{equation}\pmb{v} = \frac{1}{{{k_x}}}{\partial _x} + \frac{1}{{{k_y}}}{\partial _y}\;\notag\end{equation} Invariant found from MOC application: \begin{equation}z = {k_x}x - {k_y}y\notag\end{equation} PDE found after following the Olver steps involving the chain rule: \begin{equation}{v_{tt}} - \left( {{k_x}^2 + {k_y}^2} \right){v_{zz}} = 0\notag\end{equation} We see the “orbit” of this solution has coupled $x$ and $y$ into an invariant we called $z$. The time coordinate was not coupled.

We apply a separation of variables which produces this solution: \begin{equation}u = A\exp \left( {i\eta \left( {{k_x}x - {k_y}y} \right) + i\omega t} \right)\notag\end{equation} with the dispersion relation: \begin{equation}{\omega ^2} = {c^2}{\eta ^2}\left( {{k_x}^2 + {k_y}^2} \right)\notag\end{equation} So we have a simple plane wave-like solution with speed \begin{equation}c = \frac{\omega }{{\eta \sqrt {{k_x}^2 + {k_y}^2} }}\notag\end{equation} Any particular phase of the argument is a straight line in the $xy$ plane and propagates normal to the wavefront.


Example B: Full translation (including time): \begin{equation}\pmb{v} = \frac{1}{{{k_x}}}{\partial _x} + \frac{1}{{{k_y}}}{\partial _y} + \frac{1}{\omega }{\partial _t}\;\notag\end{equation} Invariants: \begin{equation}g = {k_x}x - \omega t\notag\end{equation} \begin{equation}h = {k_y}y - \omega t\notag\end{equation} The new PDE becomes \begin{equation}\left( {{\omega ^2} - {c^2}{k_x}^2} \right){v_{gg}} + \left( {{\omega ^2} - {c^2}{k_y}^2} \right){v_{hh}} + 2{\omega ^2}{v_{hg}} = 0\notag\end{equation} Apply and ansatz: \begin{equation}v = A\exp \left( {\eta g} \right)\exp \left( {\eta h} \right)\notag\end{equation} A solution: \begin{equation}u = A\exp \eta \left( {{k_x}x + {k_y}y - 2\omega t} \right)\notag\end{equation} Dispersion relation (necessary as part of the PDE solution): \begin{equation}{k_x}^2 + {k_y}^2 = 4{\omega ^2}/{c^2}\notag\end{equation} Note $\eta $ may be imaginary, so this could represent an oscillating propagating waveform, not unlike the spatial only translation solution above but with a specified dispersion relation (no arbitrary constant as in above).

Example C. Augmented translation: \begin{equation}\pmb{v} = \frac{1}{{{k_x}}}{\partial _x} + \frac{1}{{{k_y}}}{\partial _y} + \frac{1}{\omega }{\partial _t} + u{\partial _u}\;\;\notag\end{equation} Invariants: \begin{equation}g = {k_x}x - \omega t\notag\end{equation} \begin{equation}h = {k_y}y - \omega t\notag\end{equation} \begin{equation}v = u\exp \left( { - \omega t} \right)\notag\end{equation} PDE \begin{equation}\left( {{\omega ^2} - {c^2}{k_x}^2} \right){v_{gg}} + \left( {{\omega ^2} - {c^2}{k_y}^2} \right){v_{hh}} + 2{\omega ^2}{v_{hg}} + {\omega ^2}\left( {v - 2{v_g} - 2{v_h}} \right) = 0\notag\end{equation} Ansatz: \begin{equation}v = A\exp \left( {\eta g} \right)\exp \left( {\eta h} \right)\notag\end{equation} \begin{equation}\left[ {\left( {{\omega ^2} - {c^2}{k_x}^2} \right){\eta ^2} + \left( {{\omega ^2} - {c^2}{k_y}^2} \right){\eta ^2} + 2{\omega ^2}{\eta ^2} + {\omega ^2}\left( {1 - 2\eta - 2\eta } \right)} \right]A\exp \left( {\eta g} \right)\exp \left( {\eta h} \right) = 0\notag\end{equation} \begin{equation}\left( {{\omega ^2} - {c^2}{k_x}^2} \right){\eta ^2} + \left( {{\omega ^2} - {c^2}{k_y}^2} \right){\eta ^2} + 2{\omega ^2}{\eta ^2} + {\omega ^2}\left( {1 - 4\eta } \right) = 0\notag\end{equation} Define $\gamma $ such that \begin{equation}{k_x}^2 + {k_y}^2 = \gamma \frac{{{\omega ^2}}}{{{c^2}}}\notag\end{equation} Then to satisfy the PDE we require \begin{equation}\left( {4{\omega ^2} - \gamma {\omega ^2}} \right){\eta ^2} - 4{\omega ^2}\eta + {\omega ^2} = 0\notag\end{equation} \begin{equation}\left( {4 - \gamma } \right){\eta ^2} - 4\eta + 1 = 0\notag\end{equation} A convenient value for $\eta $ is $\eta = 1 \to \gamma = 1$, giving us this dispersion relation \begin{equation}{k_x}^2 + {k_y}^2 = \frac{{{\omega ^2}}}{{{c^2}}}\notag\end{equation} and this solution: \begin{equation}u = v\exp \left( {\omega t} \right) = A\exp \left( {\eta g} \right)\exp \left( {\eta h} \right)\exp \left( {\omega t} \right) = A\exp \left( {\left( {{k_x}x - \omega t} \right)} \right)\exp \left( {\left( {{k_y}y - \omega t} \right)} \right)\exp \left( {\omega t} \right)\notag\end{equation} \begin{equation}u = A\exp \left( {\left( {{k_x}x + {k_y}y} \right) - \omega t} \right)\notag\end{equation} Note we no longer have an arbitrary prefactor in the exponent and this is not an oscillating waveform.

Example D. Dilation: \begin{equation}\pmb{d} = x{\partial _x} + y{\partial _y} + t{\partial _t}\;\notag\end{equation} Invariants: \begin{equation}g = \frac{x}{t}\;\;{\text{and}}\;\;h = \frac{y}{t}\notag\end{equation} PDE \begin{equation}2{v_g}g + 2{v_{gh}}gh + 2{v_h}h + {v_{gg}}\left( {{g^2} - {c^2}} \right) + {v_{hh}}\left( {{h^2} - {c^2}} \right) = 0\notag\end{equation} A narrowly determined solution (found only by back-tracking the result of a joint symmetry vector application): \begin{equation}u = A\ln \left( {\frac{{{k_x}x - {k_y}y - \omega t}}{{{k_x}x - {k_y}y + \omega t}}} \right)\notag\end{equation} with dispersion relation: \begin{equation}{k_x}^2{c^2} + {k_y}^2{c^2} = {\omega ^2}\notag\end{equation} Example E. Augmented Dilation \begin{equation}d = x{\partial _x} + y{\partial _y} + t{\partial _t} + u{\partial _u}\notag\end{equation} Invariants: \begin{equation}g = \frac{x}{t}\;\;{\text{and}}\;\;h = \frac{y}{t}\;\;and\;\;f = \frac{u}{t}\notag\end{equation} PDE \begin{equation}2{f_{gh}}gh + {f_{gg}}\left( {{g^2} - {c^2}} \right) + {f_{hh}}\left( {{h^2} - {c^2}} \right) = 0\notag\end{equation} Simplistic solution: \begin{equation}u = A\left( {{k_x}x + {k_y}y + \omega t} \right)\notag\end{equation} Dispersion relation: \begin{equation}{k_x}^2{c^2} + {k_y}^2{c^2} = {\omega ^2}\notag\end{equation} Example F. Rotation only \begin{equation}{\pmb{r}_{xy}} = - y{\partial _x} + x{\partial _y}\notag\end{equation} Invariant: \begin{equation}{x^2} + {y^2} = {z^2}\notag\end{equation} PDE \begin{equation}{v_{tt}} - {c^2}{v_z}\frac{1}{z} - {c^2}{v_{zz}} = 0\notag\end{equation} Solution (via separation of variables): \begin{equation}u = A\;{J_0}\left( {k{{\left( {{x^2} + {y^2}} \right)}^{1/2}}} \right){e^{ \pm i\omega t}}\notag\end{equation} Dispersion relation: \begin{equation}{c^2}{k^2} = {\omega ^2}\notag\end{equation} This is cylindrically symmetric amplitude that oscillates in time. It does not propagate.
