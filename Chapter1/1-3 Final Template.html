Section 1.3

Continuous Transformation Groups

Our goal here is to mathematically move along the curve-path as it crosses contour lines of our function-field, sampling the field as we move along.  More specifically, we want to move along the path very slowly, making only infinitesimal steps.  We invoke a mythical creature Inchie the Inchworm (credit BGO for this silliness) and describe his movement mathematically.

For starters, we will inch along a path parallel to the x-axis, each step an infinitesimal distance $\varepsilon $.  From ${x_o}$ we progress to ${x_1} = {x_o} + \varepsilon $, then to ${x_2} = {x_1} + \varepsilon $, etc.

To enable moving along arbitrary paths we introduce an operator that, at least for the above example, generates the same result but in a way enabling generalization.
\begin{equation} {x_{new}} = \left( {1 + \varepsilon \frac{\partial }{{\partial {x_{old}}}}} \right){x_{old}} = {x_{old}} + \varepsilon \notag\end{equation}
And therefore we have something like:
\begin{equation}{x_n} = {\left[ {\left( {1 + \varepsilon \frac{\partial }{{\partial x}}} \right)} \right]^n}x{|_{x = {x_o}}}\notag\end{equation}
so for example,
\begin{equation}{x_2} = {\left[ {\left( {1 + \varepsilon \frac{\partial }{{\partial x}}} \right)} \right]^2}x{|_{x = {x_o}}} = \left( {1 + \varepsilon \frac{\partial }{{\partial x}}} \right)\left( {1 + \varepsilon \frac{\partial }{{\partial x}}} \right)x{|_{x = {x_o}}} = \left( {1 + \varepsilon \frac{\partial }{{\partial x}}} \right)\left( {x + \varepsilon } \right){|_{x = {x_o}}} = {x_o} + \varepsilon  + \varepsilon  = {x_o} + 2\varepsilon \notag\end{equation}
In general
\begin{equation}{x_n} = {x_o} + n\varepsilon  \equiv {a_n}\notag\end{equation}

To realize a finite result, $n$ must go to infinity at a rate that makes $n\varepsilon  = a$ finite.  Since ${a_n}$ is arbitrary at this point, we have realized a continuous transformation, called the translation group. This motivates us to examine an important underlying theme of continuous group theory.
