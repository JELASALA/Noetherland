2.7.2 – Translation in space and dilation

Example 2.7.2: Spatial translation in $x$ and $y$ and the dilation vector applied together: Recall this is the original EOM or PDE: \begin{equation}{u_{tt}} - {c^2}\left( {{u_{xx}} + {u_{yy}}} \right) = 0\notag\end{equation} or in units of $c = 1$: \begin{equation}{u_{tt}} - \left( {{u_{xx}} + {u_{yy}}} \right) = 0\notag\end{equation} These are the symmetry vectors we will employ: \begin{equation}\pmb{v} = \frac{1}{{{k_x}}}{\partial _x} + \frac{1}{{{k_y}}}{\partial _y}\notag\end{equation} \begin{equation}\pmb{d} = x{\partial _x} + y{\partial _y} + t{\partial _t}\notag\end{equation} We have reason to believe a solution to this combination of symmetry vectors is possible based on the commutator of the two generating vectors.

Level 1:

We first apply the displacement vector to a representative G-invariant solution $\zeta $ which requires: \begin{equation}v\left( \zeta \right) = \frac{1}{{{k_x}}}\frac{{\partial \zeta }}{{\partial x}} + \frac{1}{{{k_y}}}\frac{{\partial \zeta }}{{\partial y}} = 0\notag\end{equation} MOC procedure: \begin{equation}{k_x}dx = {k_y}dy\;\;\notag\end{equation} \begin{equation} \to {k_x}x = {k_y}y + {c_1} \to {c_1} = {k_x}x - {k_y}y \equiv {y^1}\notag\end{equation} Note on symbols used:
•
Careful not to confuse the superscripted $y$-symbols with the spatial variable $y$.
•
With MOC we use subscripted ${c_i}$ as integration constants, which are not to be confused with the constants used to express the symmetry vector generator coefficients in Appendix 2A

Level 2: The next step is to couple z and t using another (different) vector generator. We will use the dilation or scaling vector as the commutator of this vector and the translation vector returns the translation vector forming a sub-group. \begin{equation}\pmb{d} = x{\partial _x} + y{\partial _y} + t{\partial _t}\notag\end{equation} We first have to express this vector generator in terms of z and t. Operate on a placeholder function $f$: \begin{equation}\pmb{d}f = x{\partial _x}f + y{\partial _y}f + t{\partial _t}f\notag\end{equation} Apply \begin{equation}z = {k_x}x - {k_y}y\notag\end{equation} so \begin{equation}x = \frac{{z + {k_y}y}}{{{k_x}}};\;\;y = - \frac{{z - {k_x}x}}{{{k_y}}}\notag\end{equation} \begin{equation}\pmb{d}f = \left( {\frac{{z + {k_y}y}}{{{k_x}}}} \right){\partial _z}f\frac{{dz}}{{dx}} - \left( {\frac{{z - {k_x}x}}{{{k_y}}}} \right){\partial _z}f\frac{{dz}}{{dy}} + t{\partial _t}f\notag\end{equation} \begin{equation}\pmb{d}f = \left( {\frac{{z + {k_y}y}}{{{k_x}}}} \right){\partial _z}f{k_x} + \left( {\frac{{z - {k_x}x}}{{{k_y}}}} \right){\partial _z}f{k_y} + t{\partial _t}f\notag\end{equation} \begin{equation}\pmb{d}f = \left( {z + {k_y}y} \right){\partial _z}f + \left( {z - {k_x}x} \right){\partial _z}f + t{\partial _t}f\notag\end{equation} \begin{equation}\pmb{d}f = \left( {2z + {k_y}y - {k_x}x} \right){\partial _z}f + t{\partial _t}f\notag\end{equation} \begin{equation}\pmb{d}f = \left( {2z - z} \right){\partial _z}f + t{\partial _t}f\notag\end{equation} \begin{equation}\pmb{d} = z{\partial _z} + t{\partial _t}\notag\end{equation} We need a solution now for the invariance condition: \begin{equation}\pmb{d}\left( \zeta \right) = z\frac{{\partial \zeta }}{{\partial z}} + t\frac{{\partial \zeta }}{{\partial t}} = 0\notag\end{equation} MOC gives us a very simple solution with ${c_4}$ is the integration “constant”: \begin{equation}\frac{{dz}}{z} = \frac{{dt}}{t} \to \ln \left( z \right) = \ln \left( t \right) + \ln \left( {{c_4}} \right)\notag\end{equation} \begin{equation}{\text{z}} = {c_4}t\notag\end{equation} so \begin{equation}{c_4} = \frac{z}{t} \equiv w\notag\end{equation} for symbolic simplicity.
