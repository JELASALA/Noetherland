Section 1.3.2

Path to Invariance

We may ask for our Inchie translation example if there an invariance involved and/or is something conserved?  That depends on what's going on around (under?) Inchie as it moves/translates.  To realize a potential invariance we've inserted the process of movement, the most basic form of change one can imagine to discover if something varies or stays constant.

The other big takeaway here is directly from Sophus Lie — we can almost always create change looking for invariance and conservation by using infinitesimal approximations — we can stay down at the Inchie level as long as we are willing to employ that “trick” of putting an operator next to our infinitesimal. By inserting that partial derivative operator into our "motion creator" or "change generator" we can think of that partial derivative as generating Inchie's curve-path, or the flow induced, by the operator.  Indeed, Lie recommended people think about the "flow" pictures of hydrodynamics that fluids engineers (with a little help from local applied mathematicians) had developed.

To truly appreciate invariance we move to two dimensions.  We start the simplest geometrical object we can think of besides a single point: a straight line.

Now we propose a set of continuous, infinitesimal transformation equations that take us from $\left( {x,y} \right)$ to $\left( {x',y'} \right)$:

\begin{equation}x' = x - \varepsilon y\;\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;\;y' = y + \varepsilon x\notag\end{equation}

Notice these can be written in matrix form:
\begin{equation}\left( {\begin{array}{*{20}{c}}
  {x'} \\
  {y'}
\end{array}} \right) = \left( {\begin{array}{*{20}{c}}
  1&{ - \varepsilon } \\
  \varepsilon &1
\end{array}} \right) \cdot \left( {\begin{array}{*{20}{c}}
  x \\
  y
\end{array}} \right)\notag\end{equation}

And let's remember that quote from Bill Burke cited above: A symmetry is a transformation that leaves the structure essentially unchanged.

Is the proposed transformation a symmetry operation? Since we started with a straight line the operation should produce another straight line if it is some kind of symmetry operation.  Let us examine this operation on the straight line in the figure above:
\begin{equation}\left( {\begin{array}{*{20}{c}}
  {{x_2}} \\
  {{y_2}}
\end{array}} \right) = \left( {\begin{array}{*{20}{c}}
  1&{ - \varepsilon } \\
  \varepsilon &1
\end{array}} \right)\cdot \left( {\begin{array}{*{20}{c}}
  {{x_1}} \\
  {{y_1}}
\end{array}} \right)\;\notag\end{equation} where \begin{equation}{y_1} = {m_1}{x_1} + {b_1}\notag\end{equation} so \begin{equation}{x_2} = {x_1} - \varepsilon \left( {{m_1}{x_1} + {b_1}} \right) \to {x_1} = \frac{{{x_2} + \varepsilon {b_1}}}{{1 - \varepsilon {m_1}}}\notag\end{equation}

Then \begin{equation}{y_2} = \varepsilon {x_1} + {y_1} = \varepsilon \frac{{{x_2} + \varepsilon {b_1}}}{{1 - \varepsilon {m_1}}} + {m_1}\frac{{{x_2} + \varepsilon {b_1}}}{{1 - \varepsilon {m_1}}} + {b_1}\notag\end{equation}

\begin{equation}{y_2} = \frac{{\left( {{m_1} + \varepsilon } \right)}}{{1 - \varepsilon {m_1}}}{x_2} + \left( {\frac{{\left( {{m_1} + \varepsilon } \right)\varepsilon }}{{1 - \varepsilon {m_1}}} + 1} \right){b_1}\notag\end{equation}

\begin{equation}{y_2} = \frac{{\left( {{m_1} + \varepsilon } \right)}}{{1 - \varepsilon {m_1}}}{x_2} + \left( {\frac{{{\varepsilon ^2} + 1}}{{1 - \varepsilon {m_1}}}} \right){b_1} \equiv {m_2}x + {b_2}\notag\end{equation}

Treating $\varepsilon $ as infinitesimal keep the lowest order “correction” terms:

\begin{equation}{y_2} = \left[ {{m_1} + \varepsilon \left( {{m_1}^2 + 1} \right)} \right]{x_2} + \left( {1 + \varepsilon {m_1}} \right){b_1} \equiv {m_2}x + {b_2}\notag\end{equation}

To first order we have tilted the line a bit and slightly changed the intercept. Therefore, the "structure" is left "essentially unchanged".  We have a set of continuous transformation equations that left a straight line straight.  The straight line is invariant in its shape when changed by the symmetry group of the transformation equations (rotation in this case).  This statement implies the transformation is an element of a group, and intuitively you can see that it might qualify.  We can take this further.

In the spirit of Inchie’s translation let us apply the infinitesimal transformation

\begin{equation}\left( {\begin{array}{*{20}{c}}   {x'} \\   {y'} \end{array}} \right) = \left( {\begin{array}{*{20}{c}}   1&{ - \varepsilon } \\   \varepsilon &1 \end{array}} \right)\cdot \left( {\begin{array}{*{20}{c}}   x \\   y \end{array}} \right)\notag\end{equation} repeatedly, mathematically: \begin{equation}\mathop {\lim }\limits_{n \to \infty } {\left( {\begin{array}{*{20}{c}}
  1&{ - \varepsilon } \\   \varepsilon &1 \end{array}} \right)^n}\notag\end{equation}

But we have to be prepared to implement that "interior" limit of $n\varepsilon $ being finite.  So let's modify the above expression to read: \begin{equation}\mathop {\lim }\limits_{\varepsilon  \to 0;\;n \to \infty } {\left( {\begin{array}{*{20}{c}}   1&{ - \frac{{n\varepsilon }}{n}} \\   {\frac{{n\varepsilon }}{n}}&1 \end{array}} \right)^n}\notag\end{equation}

Well in the limit we assume $n\varepsilon  \equiv \theta $ becomes a finite quantity:

\begin{equation}\mathop {\lim }\limits_{n \to \infty } {\left( {\begin{array}{*{20}{c}}   1&{ - \frac{\theta }{n}} \\   {\frac{\theta }{n}}&1 \end{array}} \right)^n} = \mathop {\lim }\limits_{n \to \infty } {\left[ {\left( {\begin{array}{*{20}{c}}   1&0 \\   0&1 \end{array}} \right) + \frac{\theta }{n}\left( {\begin{array}{*{20}{c}}
  0&{ - 1} \\   1&0 \end{array}} \right)} \right]^n} \equiv \mathop {\lim }\limits_{n \to \infty } {\left[ {\mathbb{I} + \frac{{\theta \;J}}{n}} \right]^n}\notag\end{equation} with \begin{equation}\mathbb{I} \equiv \left( {\begin{array}{*{20}{c}}
  1&0 \\
  0&1
\end{array}} \right)\;\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;\;J \equiv \left( {\begin{array}{*{20}{c}}
  0&{ - 1} \\
  1&0
\end{array}} \right)\notag\end{equation}

Now the limit can be expanded suggestively:

\begin{equation}{\left[ {\mathbb{I} + \frac{{\theta \;J}}{n}} \right]^n} = \left( {\mathbb{I} + \frac{{\theta \;J}}{1}} \right)\left( {\mathbb{I} + \frac{{\theta \;J}}{2}} \right)\left( {\mathbb{I} + \frac{{\theta \;J}}{3}} \right) \ldots \left( {\mathbb{I} + \frac{{\theta \;J}}{n}} \right) = \mathbb{I} + \frac{{\theta \;J}}{1} + \frac{{\theta \;J}}{1}\frac{{\theta \;J}}{2} +  + \frac{{\theta \;J}}{1}\frac{{\theta \;J}}{2}\frac{{\theta \;J}}{3} +  \ldots \notag\end{equation}

\begin{equation}{\left[ {\mathbb{I} + \frac{{\theta \;J}}{n}} \right]^n} = \mathbb{I} + \frac{{\theta \;J}}{1} + \frac{{{{\left( {\theta \;J} \right)}^2}}}{{2!}} + \frac{{{{\left( {\theta \;J} \right)}^3}}}{{3!}} +  \ldots  + \frac{{{{\left( {\theta \;J} \right)}^n}}}{{n!}}\notag\end{equation} In the limit: \begin{equation}\mathop {\lim }\limits_{n \to \infty } {\left( {\mathbb{I} + \frac{{\theta J}}{n}} \right)^n} = \mathbb{I} + \mathop \sum \limits_{n = 1}^\infty  \frac{{{{\left( {\theta J} \right)}^n}}}{{n!}} = {e^{\theta J}}\notag\end{equation}

Computing increasing powers of the J-matrix produces a repeating pattern at every fourth power: \begin{equation}n = 1:\;\;\left( {\begin{array}{*{20}{c}} 0&{ - 1} \\ 1&0 \end{array}} \right)\notag\end{equation} \begin{equation}n = 2:\;\;\left( {\begin{array}{*{20}{c}} 0&{ - 1} \\ 1&0 \end{array}} \right)\left( {\begin{array}{*{20}{c}} 0&{ - 1} \\ 1&0 \end{array}} \right) = \left( {\begin{array}{*{20}{c}} { - 1}&0 \\ 0&{ - 1} \end{array}} \right)\notag\end{equation} \begin{equation}n = 3:\;\;\left( {\begin{array}{*{20}{c}} { - 1}&0 \\ 0&{ - 1} \end{array}} \right)\left( {\begin{array}{*{20}{c}} 0&{ - 1} \\ 1&0 \end{array}} \right) = \left( {\begin{array}{*{20}{c}} 0&1 \\ { - 1}&0 \end{array}} \right)\notag\end{equation} \begin{equation}n = 4:\;\;\left( {\begin{array}{*{20}{c}} 0&1 \\ { - 1}&0 \end{array}} \right)\left( {\begin{array}{*{20}{c}} 0&{ - 1} \\ 1&0 \end{array}} \right) = \left( {\begin{array}{*{20}{c}} 1&0 \\ 0&1 \end{array}} \right)\notag\end{equation} \begin{equation}n = 5:\;\;\left( {\begin{array}{*{20}{c}} 1&0 \\ 0&1 \end{array}} \right)\left( {\begin{array}{*{20}{c}} 0&{ - 1} \\ 1&0 \end{array}} \right) = \left( {\begin{array}{*{20}{c}} 0&{ - 1} \\ 1&0 \end{array}} \right)\notag\end{equation} So \begin{equation}\mathbb{I} + \mathop \sum \limits_{n = 1}^\infty \frac{{{{\left( {\theta J} \right)}^n}}}{{n!}} = \mathbb{I} + \theta \left( {\begin{array}{*{20}{c}} 0&{ - 1} \\ 1&0 \end{array}} \right) + \frac{{{\theta ^2}}}{{2!}}\left( {\begin{array}{*{20}{c}} { - 1}&0 \\ 0&{ - 1} \end{array}} \right) + \frac{{{\theta ^3}}}{{3!}}\left( {\begin{array}{*{20}{c}} 0&1 \\ { - 1}&0 \end{array}} \right) + + \frac{{{\theta ^4}}}{{4!}}\left( {\begin{array}{*{20}{c}} 1&0 \\ 0&1 \end{array}} \right) + \left[ {repeat\;pattern} \right]\notag\end{equation} Look at the upper left matrix element: \begin{equation}1 + 0\theta - 1\frac{{{\theta ^2}}}{{2!}} + 0\frac{{{\theta ^3}}}{{3!}} + 1\frac{{{\theta ^4}}}{{4!}} + \left[ {repeat\;pattern} \right] = 1 - \frac{{{\theta ^2}}}{{2!}} + \frac{{{\theta ^4}}}{{4!}} - \frac{{{\theta ^6}}}{{6!}} + \frac{{{\theta ^8}}}{{8!}} + \ldots \notag\end{equation} The upper right looks like: \begin{equation}0 - \theta + \frac{{{\theta ^3}}}{{3!}} - \frac{{{\theta ^5}}}{{5!}} + \frac{{{\theta ^7}}}{{7!}} + \ldots \notag\end{equation} In the limit: \begin{equation}1 - \frac{{{\theta ^2}}}{{2!}} + \frac{{{\theta ^4}}}{{4!}} - \frac{{{\theta ^6}}}{{6!}} + \frac{{{\theta ^8}}}{{8!}} + \ldots = \cos \left( \theta \right)\notag\end{equation} \begin{equation}0 - \theta + \frac{{{\theta ^3}}}{{3!}} - \frac{{{\theta ^5}}}{{5!}} + \frac{{{\theta ^7}}}{{7!}} + \ldots = - \sin \left( \theta \right)\notag\end{equation} Altogether we obtain: \begin{equation}{e^{\theta J}} = \mathbb{I} + \mathop \sum \limits_{n = 1}^\infty \frac{{{{\left( {\theta J} \right)}^n}}}{{n!}} = \left( {\begin{array}{*{20}{c}} {\cos \theta }&{ - \sin \theta } \\ {\sin \theta }&{\cos \theta } \end{array}} \right)\notag\end{equation}



The sum expressed in matrix form is:

\begin{equation}{e^{\theta J}} = \mathbb{I} + \mathop \sum \limits_{n = 1}^\infty  \frac{{{{\left( {\theta J} \right)}^n}}}{{n!}} = \left( {\begin{array}{*{20}{c}}
  {\cos \theta }&{ - \sin \theta } \\
  {\sin \theta }&{\cos \theta }
\end{array}} \right)\notag\end{equation}

This is the SO(2) rotation matrix, one representation of the rotation group.  You are welcome to test its compliance with the requirements of a group.
