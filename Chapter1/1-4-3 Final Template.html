1.4.3 LLA to the Curve-Path

Consider again the quadratic curve $\gamma \left( s \right)$ that passes through $\left( {{x_o},{y_o}} \right)$ shown in the figure below from the previous section. 


This orange arc is one of an infinite number of parabolas that cover the entire x-y plane, open towards +y, and symmetric about the y-axis.  Here are the parametric equations for the specific parabola that passes through $\left( {{x_o},{y_o}} \right)$:

\begin{equation}x\left( s \right) = s + {x_o}\;\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;\;y\left( s \right) = s\left( {s + 2{x_o}} \right) + {y_o}\notag\end{equation}

As $s \to 0$ these equations produce the point $\left( {{x_o},{y_o}} \right)$, which we are free to set to whatever we'd like.  We will use $\left( {{x_o},{y_o}} \right) = \left( {1,1} \right)$.   Eliminating the parameter between variables exhibits the parabola passing through the point of interest and the origin:

\begin{equation}y\left( x \right) = {x^2} - x_o^2 + {y_o}\notag\end{equation}

Graphically we have:


The LLA to this curve at the point of interest can be developed from a Taylor series expansion:

\begin{equation}x\left( s \right) \cong {x_o} + s{\left( {\frac{{dx}}{{ds}}} \right)_o}\;\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;\;y\left( s \right) \cong {y_o} + s{\left( {\frac{{dy}}{{ds}}} \right)_o}\notag\end{equation}

A rather unique way of looking at this is based on Burke (1980; p. 85ff), where he rewrites these equations in terms of magnified coordinates for the local neighborhood of the point $P$ at $s = 0$.:

\begin{equation}x = {x_o} + \varepsilon X\;\;\;\;{\text{and}}\;\;\;\;y = {y_o} + \varepsilon Y\;\;{\text{with}}\;\;\;s\left( \varepsilon  \right) = {s_o} + \varepsilon S\label{A}\end{equation}

Note we now use $\varepsilon $ to express (ultimately) infinitesimal progress away from the starting point.  The new variables $\left( {X,Y} \right)$ act as magnified coordinates in the following sense.  Rearranging \eqref{A}:

\begin{equation}X = \frac{{x - {x_o}}}{\varepsilon }\;\;\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;Y = \frac{{y - {y_o}}}{\varepsilon }\notag\end{equation}

As we go from $\varepsilon  = 1$ to $\varepsilon  = 1/2$ to $\varepsilon  = 1/4$ and beyond with small fixed distances $\Delta x = x - {x_o} = \varepsilon X$ and $\Delta y = y - {y_o} = \varepsilon Y$ (say, one grid block) from $\left( {{x_o},{y_o}} \right) = \left( {1,1} \right)$ the grid expands, doubling with each reduction in $\varepsilon $.  The first couple of expansions are shown below


In the limit $\varepsilon  \to 0$ we are have only the single P at$\left( {{x_o},{y_o}} \right) = \left( {1,1} \right)$ or $s = 0$ in our magnified view, suggested by the large red dot in the figures.   To connect to the essence of the curve at that point we can think of

\begin{equation}x = {x_o} + \varepsilon X\;\;\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;\;y = {y_o} + \varepsilon Y\notag\end{equation} as arising from coordinate transformation groups from $\left( {{x_o},{y_o}} \right)$ to $\left( {x\left( \varepsilon  \right),y\left( \varepsilon  \right)} \right)$ given by analytic functions: \begin{equation}x\left( \varepsilon  \right) = \chi \left( {x,{x_o},\varepsilon } \right)\;\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;\;\;y\left( \varepsilon  \right) = \zeta \left( {y,{y_o},\varepsilon } \right)\notag\end{equation} We now expand $\chi \left( {x,{x_o},\varepsilon } \right)$ and $\zeta \left( {y,{y_o},\varepsilon } \right)$ in Taylor's series about $\varepsilon  = 0$: \begin{equation}x\left( \varepsilon  \right) = {x_o} + \varepsilon {\left( {\frac{d}{{d\varepsilon }}\chi } \right)_{\varepsilon  = 0}} + O\left( {{\varepsilon ^2}} \right)..\;.\;\notag\end{equation} and \begin{equation}y\left( \varepsilon  \right) = {y_o} + \varepsilon {\left( {\frac{d}{{d\varepsilon }}\zeta } \right)_{\varepsilon  = 0}} + O\left( {{\varepsilon ^2}} \right)..\;.\notag\end{equation} generating infinitesimal transformations by ignoring second and higher order terms in$\;\varepsilon $.  These expansions explicitly make the coordinate transformation maps $\chi \left( {x,{x_o},\varepsilon } \right)$ and $\zeta \left( {y,{y_o},\varepsilon } \right)$ depend on $\varepsilon $ in such a way that as $\varepsilon  \to 0$ the maps return to "no map", i.e. no change from $\left( {{x_o},{y_o}} \right)$  If we compare these expansions to the magnification equations at the top of the page we extract: \begin{equation}X = {\left( {\frac{d}{{d\varepsilon }}\chi } \right)_{\varepsilon  = 0}}\;\;\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;\;Y = {\left( {\frac{d}{{d\varepsilon }}\zeta } \right)_{\varepsilon  = 0}}\notag\end{equation} We are still missing the connection to our parameter s.  This is accomplished by expanding the $\varepsilon $-dependent form of our parameter $s\left( \varepsilon  \right)$ \begin{equation}s\left( \varepsilon  \right) = {s_o} + \varepsilon S\;\;\notag\end{equation}

analogously to those done for the coordinates:

\begin{equation}s\left( \varepsilon  \right) = {s_o} + \varepsilon {\left( {\frac{d}{{d\varepsilon }}\Sigma } \right)_{\varepsilon  = 0}} + O\left( {{\varepsilon ^2}} \right)..\;.\notag\end{equation} We then take \begin{equation}{s_o} = 0\;\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;\;S \equiv {\left( {\frac{d}{{d\varepsilon }}\Sigma } \right)_{\varepsilon  = 0}} = S\left( {x,{x_o},y,{y_o}} \right)\notag\end{equation} for an effective \begin{equation}s\left( \varepsilon  \right) = \varepsilon S\notag\end{equation}

so “macroscopic” paremeter $s$ and the “infinitesimal” parameter $\varepsilon $ are "scaled" by

\begin{equation}S \equiv {\left( {\frac{d}{{d\varepsilon }}\Sigma } \right)_{\varepsilon  = 0}}\notag\end{equation}

If we rearrange our expansion equations into these forms:

\begin{equation}\frac{{x\left( \varepsilon  \right) - {x_o}}}{\varepsilon } = X\;\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;\;\frac{{y\left( \varepsilon  \right) - {y_o}}}{\varepsilon } = Y\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;\;\frac{{s\left( \varepsilon  \right) - {s_o}}}{\varepsilon } = S\notag\end{equation} We can still take the limits as $\varepsilon  \to 0$ on all three LHSs and be assured that through our construction, those limits will be finite as the RHS are independent of $\varepsilon $.  In the limit all three LHSs become derivatives, yielding: \begin{equation}\frac{{dx}}{{d\varepsilon }} = X\;\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;\;\frac{{dy}}{{d\varepsilon }} = Y\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;\;\frac{{ds}}{{d\varepsilon }} = S\notag\end{equation}

As these are total derivatives we can take ratios:

\begin{equation}\frac{{dx}}{{ds}} = \frac{X}{S} = {\left( {\frac{{dx}}{{ds}}} \right)_o}\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\frac{{dy}}{{ds}} = \frac{Y}{S} = {\left( {\frac{{dy}}{{ds}}} \right)_o}\label{B}\end{equation} as well as \begin{equation}\frac{{dy}}{{dx}} = \frac{Y}{X}\notag\end{equation}

so the "coordinates" aka “rate variables” $\left\{ {X,Y} \right\}$ are objectsattached to the manifold point $\left( {{x_o},{y_o}} \right)$ where $s = s\left( \varepsilon  \right) = \varepsilon S = 0$ as $\varepsilon  \to 0$.  That is, they survive the limit, yielding by their ratio the LLA of the curve-path passing through this point.  Thus the LLA to the curve $\gamma \left( s \right):\;\left\{ {x\left( s \right),y\left( s \right)} \right\}$ is given by:

\begin{equation}\vec V = \dot \gamma \left( s \right):\;\left\{ {\frac{{dx}}{{ds}},\frac{{dy}}{{ds}}} \right\}\notag\end{equation}

This means that ratios of these "rate variables" $\left\{ {X,Y,S} \right\}$ that are attached to the manifold at $\left( {{x_o},{y_o}} \right)$ provide what we are looking for -- the LLA to the curve -- a tangent vector. 

In review:  we found “magnification” variables that persist at the point $\left( {{x_o},{y_o}} \right)$ in the limit $\varepsilon  \to 0$, and whose ratio provides the LLA or tangent vector to the curve at that point.  We may apply our result to the example at hand with these parametrized equations for the curve $y\left( x \right) = {x^2} - x_o^2 + {y_o}$: \begin{equation}x\left( s \right) = s + {x_o}\;\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;\;y\left( s \right) = s\left( {s + 2{x_o}} \right) + {y_o}\notag\end{equation} On the LHSs we substitute \begin{equation}x = {x_o} + \varepsilon X\;\;\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;\;y = {y_o} + \varepsilon Y\notag\end{equation} On the RHSs we substitute \begin{equation}s = \varepsilon S\notag\end{equation} We obtain: \begin{equation}{x_o} + \varepsilon X = \varepsilon S + {x_o}\;\;\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;\;{y_o} + \varepsilon Y = \varepsilon S\left( {\varepsilon S + 2{x_o}} \right) + {y_o}\notag\end{equation}

Now we differentiate these expressions with respect to $\varepsilon $ and obtain:

\begin{equation}X = S\;\;\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;\;Y = S\left( {\varepsilon S + 2{x_o}} \right) + \varepsilon S\left( S \right)\notag\end{equation}

And finally take the limit of both expressions as $\varepsilon  \to 0$:

\begin{equation}X = S\;\;\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;\;Y = 2{x_o}S\notag\end{equation} We now employ expressions \eqref{B}: \begin{equation}{\left( {\frac{{dx}}{{ds}}} \right)_o} = \frac{X}{S} = \frac{S}{S} = 1\;\;\;\;\;\;\;\;\;\;{\text{and}}\;\;\;\;\;\;\;\;\;\;{\left( {\frac{{dy}}{{ds}}} \right)_o} = \frac{Y}{S} = \frac{{2{x_o}S}}{S} = 2{x_o}\notag\end{equation} The LLA to a curve $\gamma \left( s \right):\;\left\{ {x\left( s \right),y\left( s \right)} \right\}$ at $\left( {{x_o},{y_o}} \right) = \left( {1,1} \right)$ is given by: \begin{equation}\vec V = {\left\{ {\frac{{dx}}{{ds}},\frac{{dy}}{{ds}}} \right\}_o} = \left\{ {1,2{x_o}} \right\} = \left\{ {1,2} \right\}\notag\end{equation}

When everything is plotted on a single Euclidean grid it would look like this:


While $\vec V$ is evidently tangent to the curve-path at point P, it is not properly "in" the manifold space of $\left( {x,y} \right)$.  Rather it is in a new attached space $\left( {X,Y} \right)$ called the tangent space which we have given the new coordinate labels in the figure below.


You may imagine tangent space “hovering” over the base manifold or distinct yet attached at the point of interest.


Insisting the tangent vector and base manifold are distinct spaces, though “attached” at the point of interest, is quite important.  The quadratic curve may be a plot of position as a function of time and the tangent vector of velocity with different units than the position plot; these could simply not occupy the same space.  In mechanics problems we get away with it since the base manifold and tangent vector space are often both Euclidean   But in differential geometry we will find that only the space with the velocity vector is guaranteed Euclidean (by construction).  The base space may be a curved surface in a higher dimensional Euclidean space, with the tangent space attached at only one point, as in the figure below.


The mathematical object $\vec V$ has two representations in this new geometry.  One is a tangent vector to a curve-path and the other is a vector operator as sum of partial differential operators. It all depends on whether we want to talk about that curve-path in the manifold whose coordinates are $\left( {x,y} \right)$ or the manifold whose coordinates are $\left( {X,Y} \right)$.  As long as we stay infinitesimal both the manifold and its tangent space are (at least locally) Euclidean so we can casually ignore the difference in coordinates. A good analogy is to think of the manifold coordinates as say lengths and the tangent space coordinates as rates.
